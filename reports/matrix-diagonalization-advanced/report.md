# 행렬의 대각화 — 확장편

> 이해편 [[행렬의 대각화]] 이 "A=PDP⁻¹로 Aⁿ을 쉽게, 피보나치 일반항을 유도"까지 보여줬다면, 확장편은 **대각화 가능성의 정확한 조건(대수적·기하적 중복도), 고유분해의 의미, 대칭행렬의 직교대각화(스펙트럼 정리), 대각화 불가일 때의 조던 표준형, 그리고 선형점화식·미분방정식·마르코프·행렬지수·PCA로 뻗는 응용**을 다룬다.

## TL;DR
- 대각화는 **고유분해(eigendecomposition)** A = PDP⁻¹ 다. 이는 "A를 **고유벡터 기저에서 표현**하면 대각(스칼라 작용)이 된다"는 기저변환의 언어다 [6][국내 1].
- **가능 조건**: n×n 행렬 A가 대각화 가능 ⟺ **일차독립 고유벡터가 n개** ⟺ 모든 고유값에서 **대수적 중복도 = 기하적 중복도** [6][국내 4]. 고유값이 모두 서로 다르면 자동으로 성립한다.
- **스펙트럼 정리**: 실대칭행렬(A=Aᵀ)은 **항상 직교대각화** 가능하다 — A = QΛQᵀ, Q는 직교행렬(고유벡터가 서로 수직), Λ는 실고유값. PCA·이차형식의 토대 [국내 subprofessor][7].
- **대각화 불가**면 **조던 표준형**으로 일반화한다: A = P J P⁻¹, J는 조던 블록(대각+상부 1). 결손된 고유벡터를 일반화 고유벡터로 메운다 [9].
- **응용**: Aⁿ·비네(점화식), **행렬지수 e^{At}로 선형 ODE 풀이**, 마르코프 정상분포, PCA(공분산 대각화), 이차형식 표준화 — 전부 "**분리(decoupling)**"라는 한 아이디어 [6][8][국내 subprofessor].

## 핵심 포인트
1. **대각화 = 기저변환.** 고유벡터를 기저로 삼으면 선형사상이 대각행렬로 표현된다 [6].
2. **조건은 중복도 일치.** 대수적=기하적이면 대각화, 아니면 결손(defective) [6][국내 4].
3. **대칭이면 무조건 직교대각화**(스펙트럼 정리) — 응용의 노른자 [7][국내 subprofessor].
4. **안 되면 조던형**이 최선의 "거의 대각" 표준형 [9].
5. **모든 응용의 본질은 decoupling** — 얽힌 동역학을 독립 모드로 분해 [8].

## 1. 직관 압축 (이해편 요약)

A = PDP⁻¹ 는 "P⁻¹로 고유벡터 좌표로 갈아타 → D로 축마다 λ배 → P로 되돌리기". 그래서 Aⁿ = PDⁿP⁻¹ 로 거듭제곱이 공짜가 되고, 피보나치 점화식의 행렬 Q=[[1,1],[1,0]]을 대각화해 비네 공식 F(n)=(φⁿ−ψⁿ)/√5 를 얻었다. 확장편은 이 그림을 **언제 가능한지·안 되면 어쩌는지·어디에 쓰는지**로 정밀화한다.

## 2. 정의 · 메커니즘 — 고유분해와 기저변환

선형사상 T: ℝⁿ→ℝⁿ 을 표준기저에서 행렬 A로 나타낸다. 만약 ℝⁿ의 **기저를 A의 고유벡터 {v₁,…,vₙ}로** 바꾸면, 그 기저에서 T의 행렬표현은
```
D = diag(λ₁, …, λₙ),   A = P D P⁻¹,   P = [v₁ | v₂ | … | vₙ]
```
가 된다. 즉 **대각화 = "고유벡터 기저에서 본 A"**. P는 "고유벡터 좌표 → 표준좌표" 기저변환 행렬이고, P⁻¹은 그 역이다 [6][국내 1]. 핵심 등식은
```
A vᵢ = λᵢ vᵢ   (i=1,…,n)   ⟺   A P = P D   ⟺   A = P D P⁻¹  (P 가역일 때)
```
**P가 가역**이려면 열(고유벡터)들이 **일차독립**이어야 한다 — 이것이 §3의 조건으로 직결된다.

### 왜 "기저변환"이라는 언어가 본질인가

같은 선형사상 T도 어떤 기저에서 보느냐에 따라 행렬이 달라진다. 표준기저에서는 A라는 (대체로 지저분한) 행렬이지만, **고유벡터 기저에서는 대각행렬 D**다. 즉 대각화는 "행렬을 바꾸는" 연산이 아니라 **"보는 좌표계를 바꿔서 같은 사상을 가장 단순하게 표현"**하는 관점 전환이다. D가 대각이라는 말은 그 좌표계에서 T가 **각 축을 λᵢ배로 늘리기만 하는 스칼라 작용**으로 분해된다는 뜻이고, 이 "축마다 독립적 스칼라 작용"이 뒤에 나오는 모든 응용(거듭제곱·미분방정식·마르코프)의 계산이 쉬워지는 유일한 이유다. P⁻¹AP = D 라는 형태로도 자주 쓰는데, 이는 A와 D가 **닮음(similar)** 관계임을 강조한 표기다. 닮은 두 행렬은 같은 고유값·같은 행렬식·같은 대각합(trace)·같은 특성다항식을 공유한다 — 즉 대각화는 A의 **불변량은 보존하면서 표현만 단순화**한다 [6].

### 미니 수치 예제 — 손으로 끝까지

A = [[4, 1], [2, 3]] 를 대각화해 보자.
```
특성다항식: det(A−λI) = (4−λ)(3−λ) − 2 = λ² − 7λ + 10 = (λ−5)(λ−2)
→ 고유값 λ₁=5, λ₂=2  (서로 다름 → 대각화 확정, §3c)

λ=5: (A−5I)v=0 → [[-1,1],[2,-2]]v=0 → v₁=(1,1)
λ=2: (A−2I)v=0 → [[ 2,1],[2, 1]]v=0 → v₂=(1,−2)

P = [[1, 1],[1, −2]],  P⁻¹ = (1/−3)[[−2,−1],[−1,1]] = [[2/3, 1/3],[1/3, −1/3]]
D = diag(5, 2)
검산: P D P⁻¹ = A  ✓
```
이제 A⁵ 같은 거듭제곱도 D⁵=diag(3125, 32)만 계산해 PD⁵P⁻¹로 즉시 얻는다. 원래 A를 다섯 번 곱하는 것보다 압도적으로 싸고, 지수가 문자 n이어도 **닫힌 형식**(closed form)이 나온다는 점이 대각화의 실전 위력이다.

## 3. 왜 그런가 — 대각화 가능성의 정확한 조건

### (a) 독립 고유벡터 n개
A(n×n)가 대각화 가능 **⟺ 일차독립 고유벡터가 정확히 n개** 존재(⟺ 고유벡터들이 ℝⁿ의 기저를 이룸) [6]. 없으면 P가 가역이 아니라 대각화 불가.

### (b) 중복도 두 종류
각 고유값 λ에 대해:
- **대수적 중복도(AM)**: 특성다항식 det(A−λI)에서 (λ−λ₀)의 지수(근의 중복수).
- **기하적 중복도(GM)**: 고유공간 ker(A−λ₀I)의 차원(그 λ의 독립 고유벡터 수).
항상 **1 ≤ GM ≤ AM** 이고,
```
A 대각화 가능  ⟺  모든 고유값에서  GM = AM
```
GM < AM 인 고유값이 하나라도 있으면 **결손(defective)** 이라 대각화 불가 [6][국내 4].

### (c) 충분조건: 서로 다른 고유값
**고유값 n개가 모두 서로 다르면**(각 AM=1) 자동으로 GM=AM=1 이라 대각화된다. 서로 다른 고유값의 고유벡터는 **항상 일차독립**이기 때문이다 [국내 camposcoffee][6]. 피보나치의 Q도 φ≠ψ라 이 경우였다.

**결손 예시**: A=[[2,1],[0,2]] 는 고유값 2(AM=2)뿐인데 고유벡터는 (1,0) 방향 하나(GM=1). GM<AM 이라 대각화 불가 → §5의 조던형이 필요하다 [9].

### (d) 판정 절차 요약

실무에서 대각화 가능성은 다음 순서로 기계적으로 판정한다.
1. **특성다항식 det(A−λI)=0** 을 풀어 고유값과 각각의 **대수적 중복도(AM)**를 구한다.
2. **AM=1인 고유값**(단순근)은 자동으로 GM=1=AM 이므로 문제없다. 넘어간다.
3. **AM≥2인 고유값**만 따로 조사한다. 그 λ에 대해 **rank(A−λI)** 를 구하면 GM = n − rank(A−λI) 다.
4. 조사 대상 λ가 **하나라도 GM<AM** 이면 → 결손, **대각화 불가**(조던형으로). 전부 GM=AM 이면 → **대각화 가능**.

이 절차의 핵심은 "**중복근이 있는 고유값에서만 사고가 난다**"는 것이다. 서로 다른 고유값만 있으면(2번에서 전부 통과) 계산 없이 대각화가 보장된다. 그래서 실전에서는 "특성다항식에 중근이 있는가?"가 첫 번째 위험 신호가 된다. 참고로 **대칭행렬은 이 조사 자체가 불필요**하다 — 스펙트럼 정리(§4 Worked 3)가 중근이 있어도 항상 GM=AM을 보장하기 때문이다 [7].

## 4. 대표 응용 (Worked)

### Worked 1 — 선형 미분방정식 x' = Ax (decoupling)
연립 1계 선형 ODE **x'(t)=Ax(t)** 를 대각화로 푼다. A=PDP⁻¹ 로 두고 **y=P⁻¹x**(고유좌표)로 바꾸면
```
y' = P⁻¹x' = P⁻¹A x = P⁻¹(PDP⁻¹)x = D (P⁻¹x) = D y
```
즉 **y'ᵢ = λᵢ yᵢ** — 서로 **완전히 분리된** 스칼라 방정식이 된다. 각각 yᵢ(t)=yᵢ(0)e^{λᵢt}, 되돌리면
```
x(t) = P e^{Dt} P⁻¹ x(0) = e^{At} x(0),   e^{Dt}=diag(e^{λ₁t},…,e^{λₙt})
```
**행렬지수 e^{At}가 대각화로 계산**된다. 고유값의 실부 부호가 안정성(수렴/발산), 허부가 진동을 결정한다 — 동역학·제어의 핵심 [8][6].

**수치 예**: A=[[4,1],[2,3]](§2 예제)로 x'=Ax를 풀면 λ=5,2 이므로 일반해는 x(t)=c₁e^{5t}(1,1)+c₂e^{2t}(1,−2). 두 고유값 모두 실부가 양수라 원점은 **불안정 노드**(모든 궤적이 발산)다. 만약 어떤 고유값의 실부가 음수라면 그 모드는 시간이 지나며 사라지고, 순허수 고유값이면 그 모드는 진동(중심)한다. 이렇게 **고유값의 위치가 곧 시스템의 정성적 거동**을 결정하는 것이 선형 안정성 이론의 출발점이다. 제어공학에서 "극점 배치(pole placement)"란 결국 이 고유값들을 원하는 위치로 옮기는 설계다 [8].

### Worked 2 — 마르코프 연쇄의 정상분포
전이행렬 P(각 열 합=1)의 거듭제곱 Pⁿ이 장기 분포를 준다. 대각화 P=SΛS⁻¹ 로 Pⁿ=SΛⁿS⁻¹. 마르코프 행렬은 **고유값 λ=1을 항상 가지며 나머지 |λ|<1**(원시적일 때). n→∞에서 λ=1 항만 살아남아
```
Pⁿ → (λ=1 고유벡터로 만든 사영) = 정상분포
```
로 수렴한다. **최대 고유값이 장기 거동을 지배**하는 것은 이해편의 피보나치 φ, 그리고 [[레슬리 행렬]]의 λ₁과 **완전히 같은 원리**다 [6][국내 subprofessor].

**수치 예**: 전이행렬 P=[[0.9, 0.5],[0.1, 0.5]](열 합 1)를 보자. 고유값은 λ₁=1, λ₂=0.4다. λ=1의 고유벡터를 확률(합=1)로 정규화하면 정상분포 π=(5/6, 1/6)≈(0.833, 0.167)이다. 초기분포가 무엇이든 Pⁿ을 곱해 나가면 λ₂ⁿ=0.4ⁿ 항이 기하급수적으로 사라져 **π로 수렴**한다. 수렴 속도는 **두 번째로 큰 고유값의 크기(|λ₂|=0.4)**가 지배하는데, 이 값을 **스펙트럼 갭**이라 부르고 갭이 클수록(=|λ₂|가 작을수록) 빨리 섞인다. 구글 페이지랭크, MCMC 표본추출의 수렴 분석이 전부 이 스펙트럼 갭 논리다 [6][8].

### Worked 3 — 대칭행렬·이차형식 (스펙트럼 정리)
실대칭 A=Aᵀ 이면 스펙트럼 정리로 **A=QΛQᵀ**(Q 직교, 고유벡터 정규직교, λ 실수). 이차형식 xᵀAx 를 **주축(고유벡터 방향)** 좌표 y=Qᵀx 로 바꾸면
```
xᵀAx = yᵀΛy = λ₁y₁² + … + λₙyₙ²   (교차항 없는 표준형)
```
**타원·쌍곡선 주축, 최적화의 오목/볼록 판정(λ 부호), PCA(공분산행렬 대각화 → 주성분=고유벡터)** 가 전부 여기서 나온다. 국내 워크북도 "대각화·스펙트럼정리·양정치·PCA·저계수근사"를 한 묶음으로 다룬다 [국내 subprofessor][7].

**수치 예**: A=[[2,1],[1,2]](대칭)의 고유값은 λ=3, 1이고 고유벡터는 (1,1), (1,−1)로 **서로 수직**이다. 정규화하면 Q=(1/√2)[[1,1],[1,−1]], Λ=diag(3,1)로 A=QΛQᵀ. 이차형식 2x²+2xy+2y² 는 주축좌표에서 **3u²+v²** (교차항 xy가 사라짐)로 정리되어, 값이 항상 양수(두 고유값 모두 양수)임이 한눈에 보인다 = **양의 정부호**. 최적화에서 헤세행렬의 고유값 부호로 극소/극대/안장점을 판정하는 것, 통계에서 공분산행렬(항상 대칭·양반정부호)을 대각화해 분산이 가장 큰 방향(=최대 고유값의 고유벡터)을 첫 주성분으로 뽑는 PCA가 **모두 이 한 그림의 응용**이다. 스펙트럼 정리가 "대칭이면 고유벡터가 정규직교"를 보장하기에 Qᵀ=Q⁻¹ 이 되어 계산과 해석이 동시에 깔끔해진다 [7][국내 subprofessor].

## 5. 통합 의미 — 대각화가 안 될 때: 조던 표준형

모든 행렬이 대각화되진 않는다(§3c 결손). 그래도 **복소수 위에서는 항상 "거의 대각"인 조던 표준형**으로 만들 수 있다 [9]:
```
A = P J P⁻¹,   J = diag(J₁, …, J_k),   Jᵢ = ┌ λ 1      ┐
                                            │   λ 1    │   (조던 블록)
                                            │     ⋱ 1  │
                                            └       λ  ┘
```
- 대각에 고유값 λ, 바로 위 초대각선에 1. **결손된 고유벡터를 일반화 고유벡터**(ker(A−λI)^k)로 메운다.
- 블록 크기가 1이면 대각화 가능한 경우와 일치. **조던형은 대각화의 일반화**다.
- Aⁿ·e^{At}도 조던형으로 계산되며, 결손 고유값은 **다항식×λⁿ** 항(예: nλ^{n−1})을 낳는다 — 임계감쇠·공명 현상의 수학적 뿌리.

**일반화 고유벡터 계산 (Worked)**: §3의 결손 예 A=[[2,1],[0,2]]로 조던형을 실제로 구성해 보자. 고유값 λ=2, 고유벡터 v₁=(1,0)뿐이다(GM=1). 부족한 두 번째 기저벡터는 **(A−2I)v₂ = v₁** 을 푸는 **일반화 고유벡터**로 얻는다:
```
(A−2I) = [[0,1],[0,0]],  [[0,1],[0,0]] v₂ = (1,0)  →  v₂ = (0,1)
P = [v₁ | v₂] = [[1,0],[0,1]],  J = [[2,1],[0,2]]  →  A = P J P⁻¹
```
이 A는 이미 조던형이지만, 절차상 (A−2I)의 상(image)에 놓인 v₁을 먼저 잡고 그 원상(preimage)으로 v₂를 채운다는 점이 일반 행렬에도 그대로 적용된다. 거듭제곱은 Jⁿ=[[2ⁿ, n·2^{n−1}],[0, 2ⁿ]] 처럼 **n·2^{n−1}** 이라는 다항식×지수 항이 생기는데, 이것이 결손 고유값이 만드는 특유의 성장 패턴이다. 물리적으로는 감쇠진동에서 두 근이 겹치는 **임계감쇠**, 공진에서 진폭이 t에 비례해 커지는 **세속항(secular term)**이 정확히 이 조던 블록의 발현이다 [9][8].

정리하면: **대각화(모드 분리)가 이상적 목표, 스펙트럼 정리(대칭)가 가장 좋은 경우, 조던형이 최후의 보루**다.

## 6. 🗺️ 개념 발전 지도

기본 대각화의 가정을 하나씩 풀면 선형대수의 넓은 지형이 열린다.

| 방향 | 가정을 풀면 | 무엇이 열리나 | 어디서 쓰나 | 더 알아야 할 것 |
|---|---|---|---|---|
| **대칭·정규화** | 일반 A → 대칭/정규행렬 | **스펙트럼 정리**(직교대각화 A=QΛQᵀ), 정규행렬 | PCA·이차형식·양자역학 관측량 | 에르미트·유니터리·정규성 [7] |
| **결손 처리** | 대각화 가능 → 불가 | **조던 표준형**, 일반화 고유벡터, 최소다항식 | 임계감쇠·공명·구조해석 | 케일리-해밀턴·불변인자 [9] |
| **비정사각·수치** | 정사각 A → 임의 m×n | **특이값분해(SVD)** A=UΣVᵀ, 저계수근사 | 데이터 압축·추천·의사역행렬 | 조건수·수치안정성 [국내 subprofessor] |
| **무한차원** | 유한행렬 → 연산자 | 스펙트럼 이론, 함수의 고유함수 전개 | 미분방정식·양자역학·신호(푸리에) | 힐베르트 공간·자기수반연산자 |

## 7. 하나의 원리 — 왜 이 모든 게 "분리"인가

이해편부터 여기까지 등장한 사례를 한 줄로 꿰면 전부 **decoupling(분리)** 이다. 얽혀 있어 다루기 힘든 대상을, 고유벡터라는 "자연스러운 축"으로 갈아타 **서로 간섭하지 않는 독립 성분들의 합**으로 바꾼다:

| 대상 | 얽힌 형태 | 고유좌표에서 분리된 형태 | 지배 성분 |
|---|---|---|---|
| 거듭제곱 Aⁿ | 행렬 곱 반복 | 각 축 λᵢⁿ | 최대 \|λ\| (피보나치 φ) |
| 미분방정식 x'=Ax | 연립 ODE | y'ᵢ=λᵢyᵢ, 해 e^{λᵢt} | 실부 최대 λ (안정성) |
| 마르코프 Pⁿ | 확률 전이 반복 | λ=1 + 소멸항 λⁿ | 스펙트럼 갭 |
| 이차형식 xᵀAx | 교차항 포함 | Σλᵢyᵢ² | λ 부호 (정부호성) |
| 인구모델(레슬리) | 연령간 이동 | 각 모드 λⁿ | λ₁ (장기 성장률) |

즉 대각화가 하는 일은 언제나 **"복잡한 상호작용 = 단순한 독립 모드들의 중첩"**이라는 번역이고, 그 모드 중 **가장 큰 고유값에 대응하는 것이 장기 거동을 지배**한다. 이 한 문장이 선형대수 응용의 절반을 요약한다 [8][6].

## 8. 함정 · 체크리스트

- [ ] **대각화 판정은 중복도로**: 고유값만 보지 말고 각 λ의 **GM=AM** 확인 [6][국내 4].
- [ ] **서로 다른 고유값 = 충분조건**(필요조건 아님). 중복이어도 GM=AM이면 된다 [국내 camposcoffee].
- [ ] **대칭이면 고민 끝**: 직교대각화 보장, 고유벡터를 정규직교로 [7].
- [ ] **P의 열 순서 = D의 대각 순서** 반드시 일치.
- [ ] **복소 고유값**은 실행렬이라도 나온다(회전·진동) → 복소수에서 대각화 후 실수형으로 해석.
- [ ] **대각화 불가**를 "실패"로 보지 말 것 — 조던형이라는 정식 답이 있다 [9].
- [ ] **수치적으로는** 거의-결손 행렬이 불안정 → 실무는 SVD/슈어 분해를 선호.
