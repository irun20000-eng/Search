# 루이스-레슬리 행렬(Leslie Matrix) — 확장편

> 이해편 [[루이스-레슬리 행렬]] 이 "행렬을 곱하면 미래가 나온다"를 보여줬다면, 확장편은 **왜 최대 고유값이 성장률이고 고유벡터가 안정 분포인지(페론-프로베니우스), 특성방정식으로 λ를 푸는 법, 재생산가치(좌고유벡터), 민감도·탄력도 분석, 그리고 단계구조·확률·밀도의존으로의 확장**을 다룬다. 이해편의 직관을 전제한다.

## TL;DR
- 개체군 투영 n(t+1)=L·n(t)의 장기 거동은 **L의 스펙트럼(고유값·고유벡터)**이 지배한다. 핵심은 **최대 고유값 λ₁(성장률), 우고유벡터 w(안정 나이 분포), 좌고유벡터 v(재생산가치)** 세 가지다 [해외 6][해외 12].
- **페론-프로베니우스 정리**가 이를 보장한다: (원시적) 비음 행렬 L에는 **양수이며 절댓값이 가장 큰 고유값 λ₁이 유일**하게 존재하고, 그 고유벡터는 양수다. 그래서 n(t) ≈ c·λ₁ᵗ·w 로 수렴한다 [해외 6][해외 11].
- λ₁은 **오일러-로트카 특성방정식** Σ λ⁻ˣ lₓ mₓ = 1 (레슬리 행렬의 특성다항식과 동치)의 근으로 구한다 [해외 10].
- **민감도·탄력도(sensitivity/elasticity)**: ∂λ₁/∂aᵢⱼ = vᵢwⱼ/⟨v,w⟩. 어떤 생율(vital rate)을 바꿔야 λ가 가장 크게 오르는지 알려줘 **보전·관리 정책**의 근거가 된다 [해외 6][국내 5].
- 확장: **레프코비치(단계구조) 행렬, 확률적(환경변동) 모델, 밀도의존, 이배체·양성 모델**. 실제 생태·수산 관리의 표준 틀 [해외 12][국내 2].

## 핵심 포인트
1. **장기 거동 = 스펙트럼 문제.** Lᵗ의 거동은 고유값 크기 순서로 결정된다 [해외 6].
2. **페론-프로베니우스가 유일 지배고유값을 보장** — 그래서 "안정 분포로 수렴"이 정리로 성립 [해외 11].
3. **우고유벡터 w = 안정 나이 분포, 좌고유벡터 v = 재생산가치.** 두 벡터가 짝을 이룬다 [해외 12].
4. **민감도 = v와 w의 외적.** 정책 지렛대를 정량화 [해외 6].
5. **원시성(primitivity) 조건**이 깨지면(주기적 생율) 수렴 대신 진동 — 조건을 확인하라 [해외 11].

## 1. 직관 압축 (이해편 요약)

무리를 나이별 벡터 n으로 보고, 1년치 규칙(출산=윗줄, 생존=대각선 s)을 담은 L을 매년 곱한다. 오래 곱하면 나이 비율이 고정(안정 분포)되고 전체는 매년 λ배. 확장편은 이 "λ와 분포"가 **L의 최대 고유값과 고유벡터**임을 증명하고, 그로부터 정책 도구를 끌어낸다.

전개의 뼈대는 이렇다. **선형 반복 n(t+1)=Ln(t)의 장기 거동은 오로지 L의 스펙트럼이 결정한다**(§2)는 사실에서 출발해, 그 스펙트럼이 왜 "하나의 양수 지배고유값 + 양수 고유벡터"라는 좋은 구조를 갖는지를 **페론-프로베니우스**로 보증하고(§3a), λ를 실제로 구하는 **특성방정식·오일러-로트카**를 세우고(§3b), 좌고유벡터가 **재생산가치**임을 밝힌 뒤(§3c), 이 셋을 엮어 **민감도·탄력도**라는 정책 도구를 만든다(§4). 마지막으로 고정·결정론이라는 가정을 풀어 **단계구조·확률·밀도의존**으로 확장한다(§6). 즉 "행렬 하나 → 스펙트럼 → 정책 → 현실화"의 사다리다.

## 2. 정의 · 스펙트럼 분해

n×n 레슬리 행렬:
```
      ┌ f₁ f₂ … f_{n-1} f_n ┐
      │ s₁  0  …    0     0 │
  L = │  0 s₂  …    0     0 │      fᵢ ≥ 0 (출산율), 0 < sᵢ ≤ 1 (생존율)
      │  ⋮      ⋱             │
      └  0  0  … s_{n-1}   0 ┘
```
L이 대각화 가능하면 고유쌍 (λᵢ, wᵢ)로
```
n(t) = Lᵗ n(0) = Σᵢ cᵢ λᵢᵗ wᵢ
```
로 분해된다. **|λ₁| > |λ₂| ≥ …** 이면 큰 t에서 첫 항이 지배:
```
n(t) ≈ c₁ λ₁ᵗ w₁      (t → ∞)
```
따라서 **성장률 = λ₁, 안정 나이 분포 = w₁**(정규화한 우고유벡터). 다른 항들은 (λᵢ/λ₁)ᵗ → 0 으로 **과도상태(transient)**로 사라진다 [해외 6][해외 12]. 이것이 이해편 "오래 굴리면 비율이 고정"의 정확한 이유다.

이 분해를 다시 뜯어보면 **왜 초기 조건이 잊히는지**가 선명해진다. 아무리 이상한 초기 분포 n(0)로 시작해도, 그것을 고유벡터들의 합 Σcᵢwᵢ로 쓸 수 있고, 시간이 지나면 λ₁ᵗ가 가장 빨리 커지므로 **w₁ 성분만 살아남는다**. 나머지 성분은 상대적으로 (λᵢ/λ₁)ᵗ 비율로 쪼그라든다. 특히 **두 번째로 큰 고유값 λ₂**가 중요한데, |λ₂/λ₁|가 1에 가까우면 소멸이 느려 **안정화까지 오래 걸리고**, 0에 가까우면 **몇 세대 만에 안정 분포에 안착**한다. 이 비율의 로그 역수 −1/ln|λ₂/λ₁| 를 **감쇠시간(damping time)**이라 하며, "단기 예측에서 과도상태를 무시해도 되는가"를 판단하는 척도다. 이해편이 "오래 굴리면"이라고 뭉뚱그린 그 "오래"의 길이가 바로 λ₂/λ₁로 정량화된다 [해외 12][해외 8].

## 3. 왜 그런가 — 페론-프로베니우스와 특성방정식

### (a) 지배고유값의 존재·유일성 (페론-프로베니우스)
L은 **비음(nonnegative)** 행렬이고, 모든 sᵢ>0 이며 적어도 두 인접 fᵢ가 양수이면 **원시적(primitive)**이다. 페론-프로베니우스 정리에 의해:
- **양의 실수 고유값 λ₁ > 0 이 존재**하고 **절댓값이 다른 모든 고유값보다 크다**(유일 지배).
- 그에 대응하는 **우고유벡터 w₁ > 0, 좌고유벡터 v₁ > 0**(성분이 모두 양수).

원시성이 "장기적으로 하나의 성장률·하나의 안정 분포로 수렴"을 수학적으로 보증한다. **원시성이 깨지면**(예: 오직 한 나이대만 번식하는 반복산란/semelparous → 순환행렬적 구조) 지배고유값이 복소 켤레와 크기가 같아져 **수렴 대신 영구 진동**이 나타난다 [해외 11][국내 5].

원시성을 판별하는 실용 기준은 이렇다. 레슬리 행렬이 원시적일 **필요충분조건**은 "**번식하는 나이들(fᵢ>0인 i)의 최대공약수가 1**"이라는 것이다. 예컨대 나이 2와 3에서 번식하면 gcd(2,3)=1 이라 원시적이고 수렴한다. 반면 **오직 나이 3에서만** 번식하는 매미 같은 종은 gcd=3 이라 원시성이 깨져, 세 개의 세대 코호트가 서로 섞이지 않고 **3년 주기로 진동**한다(주기매미의 대발생이 이 구조와 관련된다). 이처럼 "언제 번식하느냐"의 정수론적 성질이 개체군이 **매끈히 수렴할지 영원히 출렁일지**를 가른다 — 생태학과 수론이 만나는 흥미로운 지점이다 [해외 11][해외 8].

### (b) 특성방정식 — λ를 실제로 푼다
det(L − λI)=0 을 전개하면 레슬리 행렬 특유의 형태가 나온다. 생존율 곱 lₓ = s₁s₂…s_{x-1} (x나이까지 생존확률)을 쓰면 **오일러-로트카 방정식**과 동치:
```
Σ_{x=1}^{n}  λ^(−x) · lₓ · mₓ  =  1
```
(mₓ = x나이의 출산율.) 좌변은 λ에 대해 **단조감소**하므로 **양의 실근 λ₁이 유일**하다. λ=1을 대입한 값
```
R₀ = Σ lₓ mₓ   (순재생산율, 한 세대당 새끼 수)
```
이 1보다 크면 λ₁>1, 작으면 λ₁<1이다 — **R₀와 λ₁이 같은 편에 선다**(세대당 vs 연당 성장). 국내 생태 자료의 "순재생산율" 개념이 바로 이것이다 [국내 2][해외 10].

### (c) 좌고유벡터 = 재생산가치
v₁ L = λ₁ v₁ 을 만족하는 **좌고유벡터 v₁**의 성분 vₓ는 **"x나이 개체가 앞으로 개체군에 기여할 기대 새끼 수"**, 즉 **재생산가치(reproductive value)**다. 갓 태어난 개체보다 **막 번식 적령기에 든 개체의 v가 가장 높다**(앞으로 낳을 게 많으므로). 이는 보전에서 "어느 나이를 지킬까"의 핵심 지표다 [해외 12].

## 4. 대표 응용 (Worked)

### Worked 1 — 2×2로 λ와 안정 분포 손계산
```
L = ┌ 1  2 ┐     (f₁=1, f₂=2, s₁=0.5)
    └ .5 0 ┘
특성방정식: det(L−λI) = (1−λ)(−λ) − 2·0.5 = λ² − λ − 1 = 0
⇒ λ = (1 ± √5)/2 ⇒ λ₁ = 1.618 (황금비!), λ₂ = −0.618
```
λ₁=1.618>1 이므로 **연 61.8% 성장**. 안정 분포는 (L−λ₁I)w=0 에서 w ∝ (λ₁, s₁)ᵀ = (1.618, 0.5) ⇒ 정규화하면 약 (0.76, 0.24). 즉 **성체 대비 새끼 비율 76:24로 고정**된다. (참고: f=s=1인 특수한 경우 특성방정식이 λ²=λ+1 이 되어 **피보나치·황금비**가 나온다 — 레슬리 모델과 피보나치의 연결.) [해외 9][국내 1]

### Worked 1.5 — 3×3 완전 계산과 안정 분포 수렴
이해편의 무리(f=(0,1,2), s=(0.5,0.4))로 특성방정식을 세워보자.
```
L = ┌ 0  1  2 ┐
    │ .5 0  0 │      det(L−λI)=0:
    └ 0 .4  0 ┘      −λ³ + 0.5·1·λ + 0.5·0.4·2 = 0
                     ⇒ λ³ − 0.5λ − 0.4 = 0
```
수치해를 구하면 실근 **λ₁ ≈ 0.949**(<1)와 복소 켤레근 한 쌍(|·|<λ₁)이 나온다. 즉 **연 약 5% 감소**로, 이해편에서 합계가 35→34→32로 줄던 관찰과 정확히 일치한다. 안정 분포는 w ∝ (λ₁², λ₁·s₁... ) 꼴로 계산되며, 정규화하면 대략 (0.60, 0.32, 0.13) — **새끼가 60%를 차지하는 프로필**로 고정된다. 만약 s₀를 0.5→0.7로 올리면 λ₁이 1을 넘어 **증가로 전환**된다. 이렇게 "어느 생율을 얼마나 바꾸면 λ가 1을 넘나"를 푸는 것이 관리의 출발점이고, 그 감도를 정량화한 것이 다음의 민감도다 [해외 9][해외 10].

### Worked 2 — 민감도·탄력도로 정책 고르기
λ₁의 행렬성분 aᵢⱼ에 대한 **민감도**:
```
sᵢⱼ = ∂λ₁/∂aᵢⱼ = vᵢ wⱼ / ⟨v, w⟩        (v,w는 좌·우 지배고유벡터)
```
**탄력도(비례 민감도)**는
```
eᵢⱼ = (aᵢⱼ/λ₁)·sᵢⱼ,   Σᵢⱼ eᵢⱼ = 1
```
탄력도는 "그 생율을 1% 바꾸면 λ가 몇 % 바뀌나"라, **서로 다른 단위의 출산율·생존율을 공정 비교**한다. 바다거북 사례에서 **성체 생존율의 탄력도가 알 생산의 탄력도보다 훨씬 커서**, "성체를 지켜라"가 최적으로 나온다(이해편에서 예고). 이처럼 민감도·탄력도는 **한정 자원 배분의 정량 근거**다 [해외 6][국내 5].

### Worked 3 — 확률적 성장률(환경변동)
현실의 생율은 매년 변한다. 좋은 해의 행렬 L_good와 나쁜 해 L_bad가 무작위로 번갈아 온다면, t년 뒤 개체수는 **행렬들의 곱** n(t)=L_{t}…L_{2}L_{1}n(0)로 주어진다. 이때 장기 성장률은 개별 λ의 산술평균이 아니라 **확률적 성장률** log λₛ = lim (1/t)·E[log ‖n(t)‖] 로 정의된다(랜덤 행렬곱의 리아푸노프 지수). 핵심 결과: **변동은 성장을 깎는다.** 옌센 부등식에 의해 log의 오목성 때문에 λₛ ≤ (기하평균) ≤ 산술평균이 되어, **평균 생율이 같아도 변동이 크면 개체군이 더 위태롭다**. 그래서 멸종위험 평가(MVP·PVA)에서는 λ₁ 하나가 아니라 **변동성까지** 본다 [국내 2][해외 12]. 이는 "평균만 좋으면 된다"는 직관을 정면으로 뒤집는 중요한 통찰이다.

## 5. 통합 의미 — 하나의 스펙트럼이 모든 걸 말한다

- **λ₁**: 장기 성장률(운명). >1 번성 / =1 평형 / <1 소멸.
- **w₁**(우): 안정 나이 분포(무리가 결국 갖는 모양).
- **v₁**(좌): 재생산가치(각 나이의 미래 기여).
- **감쇠비 λ₂/λ₁**: 안정 분포로 **얼마나 빨리** 수렴하는지(과도상태 길이).
- **민감도 v⊗w**: 어디를 건드려야 λ가 움직이나(정책 지렛대).

즉 **행렬 하나의 고유구조에 개체군의 과거·현재·미래·정책이 전부 인코딩**돼 있다. 이것이 행렬 개체군 모델이 생태·수산·인구학의 공용어가 된 이유다 [해외 12][국내 3].

**인구학과의 연결**도 짚어둘 만하다. 사람 인구도 같은 틀로 다룬다: fᵢ는 연령별 출산율, sᵢ는 연령별 생존율이고, λ₁은 인구가 장기적으로 늘지 줄지를 말한다. 한 사회의 합계출산율이 대체수준(선진국 약 2.1) 아래로 내려가면 순재생산율 R₀<1 이 되어 λ₁<1, 즉 **장기적 인구 감소**가 예정된다 — 한국의 저출생 논의가 정확히 이 지점이다. 다만 실제 인구는 이민·기대수명 변화·출산 시기 이동 같은 요인이 겹쳐, 단순 레슬리 투영은 **"현 생율이 유지된다면"의 사고실험**으로 읽어야 한다. 그럼에도 "연령 구조가 미래를 어떻게 관성적으로 끌고 가는가"(인구 모멘텀 — 출산율이 즉시 대체수준으로 회복돼도 관성으로 한동안 인구가 계속 변하는 현상)를 이해하는 데 이 스펙트럼 관점은 필수다 [국내 3][해외 12].

또 하나, **재생산가치 v와 안정 분포 w의 짝**은 깊은 대칭을 이룬다. w는 "미래에 각 나이가 **얼마나 많아질지**"(양), v는 "각 나이가 미래에 **얼마나 기여할지**"(질)를 말한다. 둘의 곱 v⊗w가 민감도를 이루는 것은 우연이 아니다 — **정책 효과 = (그 나이의 기여도) × (그 나이의 규모)**라는 자연스러운 분해다. 이 대칭은 선형대수의 좌·우 고유벡터 이중성이 생태학적 의미로 번역된 아름다운 예다 [해외 12].

## 6. 🗺️ 개념 발전 지도

기본 레슬리는 "나이 구조·고정 생율·밀도 무관·암컷만"이라는 네 단순화 위에 서 있다. 각 방향은 그 가정을 하나씩 풀며 현실로 다가간다 — 아래 표를 학습 로드맵으로 삼으면 된다.

| 방향 | 가정을 풀면 | 무엇이 열리나 | 어디서 쓰나 | 더 알아야 할 것 |
|---|---|---|---|---|
| **구조 일반화** | 나이 → 발달단계·크기 | **레프코비치 행렬**(대각·역행 전이 허용), 정체·역행 | 식물·산호·어류(나이 불명) | 단계전이 추정·군집화 [해외 12] |
| **확률화** | 고정 생율 → 환경변동 | 확률적 성장률 log λₛ, 소멸확률, 개체 확률성 | 멸종위험·MVP(최소존속개체군) | 랜덤행렬곱·리아푸노프 지수 [국내 2] |
| **비선형화** | 밀도 무관 → 밀도의존 | 로지스틱형 평형·주기·카오스 | 자원경쟁·수산 어획 | 분기이론·안정성(고유값<1) [국내 5] |
| **양성·공간** | 암컷만 → 양성/이배체·공간 | 짝짓기함수, 메타개체군·확산 | 성비 편향·서식지 연결성 | 이배체 동역학·분산행렬 [해외 12] |

## 7. 함정 · 체크리스트

- [ ] **원시성 확인**: 반복산란·단일번식 구조면 지배고유값 유일성이 깨져 진동 → 스펙트럼 전체를 보라 [해외 11].
- [ ] **암컷·성비 가정**: 기본형은 암컷만. 성비 편향 시 양성 모델 필요 [해외 12].
- [ ] **투영 ≠ 예측**: L은 "현재 생율이 유지되면"의 투영이다. 미래 생율 변화는 별도 [해외 6].
- [ ] **시간·나이 간격 일치**: 투영 간격과 나이대 폭이 같아야(연 단위면 1년 나이대) [해외 10].
- [ ] **민감도 vs 탄력도**: 단위 다른 생율 비교엔 **탄력도**를 써라 [해외 6].
- [ ] **과도상태 무시 금지**: 감쇠비가 1에 가까우면 안정화까지 오래 — 단기 예측은 스펙트럼만으론 부족 [해외 12].
- [ ] **음수·>1 생율 금지**: fᵢ≥0, 0<sᵢ≤1. 데이터 오류 점검.
- [ ] **생율 추정의 불확실성 전파**: 현장 관측으로 얻은 fᵢ·sᵢ에는 오차가 있다. λ₁의 신뢰구간을 부트스트랩 등으로 함께 보고해야, "λ=0.98이니 감소"라는 결론이 통계적으로 유의한지 판단할 수 있다 [해외 12].
- [ ] **투영 간격의 산란 시점(census timing)**: 번식 전 조사(pre-breeding)냐 후 조사(post-breeding)냐에 따라 fᵢ에 첫해 생존율을 곱해 넣는지가 달라진다. 같은 종이라도 행렬 성분 정의가 바뀌므로 문헌 비교 시 주의 [해외 10].

이 체크리스트의 저변에 깔린 원칙은 하나다: **레슬리 행렬은 "가정을 명시한 사고 도구"이지 수정구슬이 아니다.** 각 성분이 어떤 가정 아래 어떤 데이터로 추정됐는지를 분명히 할 때만 λ와 민감도의 해석이 신뢰를 얻는다. 모델의 가치는 정확한 예측보다도, **"어떤 가정을 바꾸면 결론이 뒤집히는가"를 투명하게 드러내는 데** 있다.
