# 행렬과 다변수 미적분 — 이해편(입문·딥)

> 📚 **이 글의 목표**: 두 주제를 *납득*시키는 것. 그림·비유·작은 숫자 예제로 "왜 이렇게 하는지"가 손에 잡히게. 엄밀한 정의·증명·확장은 짝 문서 **[[행렬과 다변수 미적분 확장편]]**에 있어요.

## TL;DR
- **행렬**은 숫자를 직사각형으로 적은 '숫자 표'예요(가로=행, 세로=열, 칸=성분) [1]. 그런데 더 중요한 건, 행렬이 사실은 **무언가를 바꾸는 '변환'**이라는 점이에요.
- 덧셈은 같은 자리끼리지만, **곱셈은 '앞의 행 × 뒤의 열'**이라는 특별한 규칙을 써요(그리고 `AB≠BA`) [2][7].
- **2×2 행렬식 `ad−bc`**는 그 변환이 **넓이를 몇 배로 바꾸는지**를 알려줘요(0이면 못 되돌림) [4][3].
- **다변수 함수**(예: 산 높이 `z=f(x,y)`)의 기울기는 **편미분**으로 재요 — "한 변수만 바꾸고 나머지는 잠깐 상수로" [5][9].

## 핵심 포인트
1. 행렬 = 숫자 표 = (더 깊게는) 공간을 바꾸는 변환 [1].
2. 곱셈은 행×열, 순서가 중요(`AB≠BA`) [2][7].
3. 행렬식 = 넓이 배율, 0이면 못 되돌림 [4].
4. 편미분 = 한 방향만 보고 잰 경사 [5].

## 본문

### 1. 행렬은 '숫자 표'… 그런데 사실은 '변환'
점수표를 떠올려 봐요:
```
        국어  영어  수학
 철수 [  90    80    70 ]
 영희 [  60   100    85 ]
```
숫자를 직사각형으로 적은 게 **행렬**이에요 — 가로줄=**행**, 세로줄=**열**, 칸 하나=**성분**, 크기는 "행×열"(위는 2×3) [1].

여기까진 단순한 표지만, 행렬의 진짜 매력은 **그림을 바꾸는 도구**라는 데 있어요. 예를 들어 평면의 모든 점에 어떤 2×2 행렬을 곱하면, 그림 전체가 **늘어나거나, 회전하거나, 기울어져요**. 즉 행렬은 "공간을 어떻게 바꿀지 적어 둔 설명서"예요. (이 관점이 다변수 미적분과 만나는 다리예요 — 끝에서 봅니다.)

### 2. 덧셈은 쉽고, 곱셈은 '짝짓기'
같은 크기 행렬의 **덧셈**은 같은 자리끼리 더하면 끝이에요 [1]:
`[[1,2],[3,4]] + [[5,6],[7,8]] = [[6,8],[10,12]]`.

**곱셈**은 칸끼리가 아니라 **"앞 행렬의 한 행"과 "뒤 행렬의 한 열"을 곱해서 더해요** [2][7]. 한 칸을 만들 때마다 두 줄의 숫자를 짝지어 합치는 거예요:
- `[[1,2],[3,4]] × [[5,6],[7,8]]`의 (1행,1열) = `1×5 + 2×7 = 19`.
- 전부 채우면 `[[19,22],[43,50]]`.

> ⚠️ **순서가 중요**: 행렬은 보통 `A×B ≠ B×A`예요 [7]. 왜냐면 곱셈은 "변환을 잇따라 적용"하는 것과 같은데, **순서를 바꾸면 결과가 달라지기** 때문이에요(예: "오른쪽으로 기울이고 회전" ≠ "회전하고 기울이기"). 직접 위 예제의 순서를 바꿔 계산하면 다른 값이 나와요.

### 3. 행렬식 — 변환이 넓이를 몇 배로 바꾸나
2×2 행렬 `[[a,b],[c,d]]`의 **행렬식**은 `ad − bc`예요 [4]. 이 숫자의 의미가 멋져요:
- **행렬식의 크기 = 그 변환이 넓이를 바꾸는 배율.** 예를 들어 행렬식이 3이면 도형 넓이가 3배가 돼요.
- 예: `[[2,0],[0,2]]`(모든 걸 2배 확대)의 행렬식은 `2×2−0=4` → 넓이가 4배(가로 2배 × 세로 2배). 직관과 맞죠.
- **행렬식이 0이면** 변환이 평면을 선(線)이나 점으로 납작하게 눌러버려서 **되돌릴 수 없어요**(역행렬이 없음) [4][3].

> 💡 이 "행렬식 = 넓이 배율"이 바로 [[야코비안 변환 이해편]]의 씨앗이에요. 야코비안은 이 아이디어를 휜 변환에 점마다 적용한 것이거든요.

### 4. 다변수 함수 — 입력이 여러 개
지금까지의 함수는 `y=f(x)`처럼 입력이 하나였죠. 그런데 현실은 여러 가지가 동시에 작용해요. 산의 **높이**는 위치 `(x, y)` 두 개로 정해져요 → `z=f(x,y)`. 이런 함수의 그래프는 평면 위의 **언덕(곡면)**처럼 생겼어요.

### 5. 편미분 — "한 방향만 보고 경사 재기"
언덕 위 한 점의 경사는 **어느 방향으로 가느냐에 따라 달라요**(동쪽은 오르막, 북쪽은 평지일 수 있죠). 그래서 한 번에 하나씩 봅니다 — 이게 **편미분**이에요:
> 관심 있는 변수 **하나만** 변화시키고, 나머지는 잠깐 **상수**로 두고 미분 [5][6][9].

기호는 `∂`(동글한 d)를 써요. `∂z/∂x`는 "`y`는 고정, `x`만 움직였을 때 높이가 변하는 비율" = **동쪽으로 갈 때의 경사**예요 [9].

작은 예제로 손에 익혀요:
- `f(x,y) = x² + 3y` → `x`로 편미분하면(`3y`는 상수) `∂f/∂x = 2x`; `y`로 편미분하면(`x²`는 상수) `∂f/∂y = 3`.
- `f(x,y) = x²y` → `∂f/∂x = 2xy`(`y`는 상수 취급), `∂f/∂y = x²`(`x²`는 상수 취급).

### 6. 두 주제가 만나는 다리
편미분으로 잰 경사들을 **여러 개 모으면**, 그게 바로 §1에서 말한 **행렬**이 돼요. 즉 "다변수 함수의 기울기"는 숫자 하나가 아니라 **편미분으로 채운 행렬**이고, 행렬이 변환이었듯 이 행렬도 "그 점 근처에서 함수가 공간을 어떻게 바꾸는지"를 나타내요. 이게 다음 개념 [[야코비안 변환 이해편]]의 출발점이에요. **행렬(그릇)과 편미분(재료)은 한 팀**이라는 게 이 글의 핵심입니다.

## 한눈 정리
- 행렬 = 숫자 표 = 공간을 바꾸는 변환.
- 곱셈은 행×열, `AB≠BA`.
- 행렬식 = 넓이 배율(0이면 못 되돌림).
- 편미분 = 한 방향만 보고 잰 경사; 모으면 행렬이 된다.

## 더 깊이 — 확장편으로
- 🌉 **행렬=선형사상의 엄밀한 정의, 가역의 삼위일체(det·rank·역행렬), 전미분=야코비, 연쇄법칙, 그리고 추상 선형사상·프레셰 미분·다양체·자동미분으로 뻗는 「개념 발전 지도」** → [[행렬과 다변수 미적분 확장편]]
