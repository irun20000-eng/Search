# 루이스-레슬리 행렬(Leslie Matrix) — 이해편

> 목표: "나이별로 출산율·생존율이 다른 동물 무리가 **몇 년 뒤 몇 마리가 될지**를 행렬 하나로 예측하는 법"을 고등학생 눈높이로 납득시키기. 엄밀한 고유값 해석·증명·일반화는 확장편 [[루이스-레슬리 행렬 확장편]] 으로.

## TL;DR
- **루이스-레슬리 행렬**은 동물·사람 무리를 **나이 그룹으로 나눠**, "**1년 뒤 각 나이대에 몇 마리가 있을지**"를 계산하는 행렬이다. 생태학·인구학의 기본 도구다 [해외 6][국내 3].
- 원리는 딱 두 줄이다: **(1) 새끼는 각 나이대의 어미가 낳고(출산율), (2) 살아남은 개체는 다음 나이대로 한 칸 이동한다(생존율).** 이걸 행렬 L에 담는다 [해외 8].
- 예측은 곱셈 한 번: **다음 해 개체수 벡터 = L × 올해 벡터**. 이걸 반복하면 몇 년 뒤든 예측된다 [해외 6][국내 1].
- 오래 반복하면 무리는 **"안정 나이 분포"**(각 나이대 비율이 고정된 상태)로 수렴하고, 전체 수는 매년 **일정 배율 λ(람다)**로 곱해진다 — 이 λ가 **개체군 성장률**이다 [해외 6].
- **λ>1이면 늘고, λ=1이면 유지, λ<1이면 결국 멸종**한다. 이 하나의 숫자가 "이 종이 번성하나 사라지나"를 말해준다 [해외 6][국내 2].

## 핵심 포인트
1. **나이로 쪼갠다.** "전체 몇 마리"가 아니라 "0살 몇, 1살 몇, 2살 몇"으로 본다(벡터) [해외 8].
2. **행렬 = 1년치 규칙.** 맨 윗줄은 **출산율**, 그 아래 대각선은 **생존율**이다 [해외 8].
3. **예측 = 행렬 곱.** L을 곱할 때마다 1년이 흐른다 [국내 1].
4. **λ(최대 고유값) = 장기 성장률.** 무리는 결국 매년 λ배로 자란다 [해외 6].
5. **λ 하나로 운명이 갈린다.** >1 번성 / =1 유지 / <1 쇠퇴 [해외 6][국내 2].

## 1. 왜 이런 게 필요한가 — "전체 숫자"만으론 부족하다

토끼 무리가 100마리 있다고 하자. 내년엔 몇 마리가 될까? 단순히 "출산율을 곱하면 되지 않나?" 싶지만, 문제가 있다. **갓 태어난 새끼와 다 큰 어른은 역할이 다르다.** 새끼는 아직 새끼를 못 낳고, 늙은 개체는 곧 죽는다. 그러니 **"전체 100마리"만 알아선 예측이 안 되고, "어느 나이가 몇 마리인지"를 알아야 한다** [해외 8][국내 3].

여기서 나온 것이 레슬리 행렬이다. 1940년대에 **E. G. 루이스(1942)**와 **P. H. 레슬리(1945)**가 각각 제안해서 "**루이스-레슬리 행렬**"이라 부른다(흔히 줄여 레슬리 행렬). 아이디어는 **"나이별로 나눠서 관리하자"**는 것이다. 이렇게 하면 멸종 위기종 보전, 어업 자원 관리, 인구 추계 같은 실전 문제를 **숫자로** 다룰 수 있다 [해외 6].

배울 점: **복잡한 무리도 "나이 그룹 + 각 그룹의 규칙(출산·생존)"으로 쪼개면 행렬 곱셈이라는 단순한 계산**으로 미래를 그릴 수 있다.

## 2. 무리를 벡터로 — 나이별로 세기

무리를 나이대로 나눠 각 칸에 개체수를 적으면 **벡터**가 된다. 예를 들어 나이대를 셋(0살·1살·2살)으로 나누면:

```
       ┌      ┐
       │ 20   │  ← 0살 (새끼) 20마리
n(올해)= │ 10   │  ← 1살 (청년) 10마리
       │  5   │  ← 2살 (어른) 5마리
       └      ┘
```

이 벡터가 **무리의 "나이 프로필"**이다. 우리의 목표는 "1년 뒤 이 벡터가 어떻게 바뀌는가"를 구하는 것이다.

## 3. 행렬 = 1년 동안 일어나는 일

1년 사이 무리에 일어나는 일은 딱 두 종류다 [해외 8][국내 3].

- **출산**: 각 나이대의 개체가 새끼(0살)를 낳는다. 나이대별 **출산율**을 f₀, f₁, f₂라 하자(보통 어린 새끼는 못 낳으니 f₀=0에 가깝다).
- **생존·성장**: 살아남은 개체가 **한 칸 위 나이대로 이동**한다. 0살→1살 생존율 s₀, 1살→2살 생존율 s₁.

이 규칙을 행렬 L에 담으면:

```
       ┌               ┐
       │ f₀  f₁  f₂    │  ← 맨 윗줄: 출산율 (모든 나이가 새끼 기여)
   L = │ s₀   0   0    │  ← s₀: 0살 중 살아남아 1살 되는 비율
       │  0  s₁   0    │  ← s₁: 1살 중 살아남아 2살 되는 비율
       └               ┘
```

**모양의 규칙**은 외우기 쉽다: **맨 윗줄 = 출산율, 바로 아래 대각선 = 생존율, 나머지는 0.** 윗줄은 "새끼가 어떻게 생기나", 대각선은 "각자 한 살 더 먹으며 얼마나 살아남나"를 말한다.

## 4. 예측 = 행렬 곱 (해마다 L을 곱한다)

이제 마법의 순간이다. **내년 벡터 = L × 올해 벡터.**

구체적 숫자로 해보자. 출산율 f₀=0, f₁=1, f₂=2, 생존율 s₀=0.5, s₁=0.4라 하고, 올해 n=(20,10,5)ᵀ.

```
       ┌ 0  1  2 ┐ ┌ 20 ┐   ┌ 0·20 + 1·10 + 2·5 ┐   ┌ 20 ┐
L·n =  │ .5 0  0 │ │ 10 │ = │ 0.5·20            │ = │ 10 │
       │ 0 .4  0 │ │  5 │   │ 0.4·10            │   │  4 │
       └         ┘ └    ┘   └                   ┘   └    ┘
```

**해석**:
- 0살 = 20 → 어른들이 낳은 새끼: 1살 10마리가 각 1마리(=10), 2살 5마리가 각 2마리(=10) → 합 20마리.
- 1살 = 10 → 작년 0살 20마리 중 절반 생존(0.5×20).
- 2살 = 4 → 작년 1살 10마리 중 40% 생존(0.4×10).

즉 **곱셈 한 번으로 무리 전체가 1년 늙고, 태어나고, 죽는 걸 동시에 처리**했다. 2년 뒤는 L을 또 곱하면 된다: n(2년) = L·(L·n) = L²·n. **10년 뒤가 궁금하면 L¹⁰을 곱한다.** 이게 레슬리 행렬의 힘이다 [국내 1][해외 6].

여기서 잠깐, **행렬 곱셈이 왜 이렇게 딱 맞아떨어지는지** 음미해보자. 행렬 곱은 원래 "각 출력 = 입력들의 가중합"을 한 번에 계산하는 장치다. 레슬리 행렬에서 **각 줄은 하나의 질문에 답한다**: 맨 윗줄은 "내년 0살은 몇 마리?"(→ 모든 나이의 출산 기여를 더함), 둘째 줄은 "내년 1살은 몇 마리?"(→ 올해 0살의 생존), 셋째 줄은 "내년 2살은?"(→ 올해 1살의 생존). 이렇게 **"1년 동안 벌어지는 모든 이동·탄생·사망"을 표 하나에 정리**해두면, 곱셈이 알아서 전부 합산해준다. 사람이 나이별로 일일이 계산할 필요가 없다. 이것이 "복잡한 동태를 행렬로 압축한다"는 말의 진짜 의미다 [해외 8][국내 행렬].

## 5. Worked Example — 3년 굴려보고 성장률 느끼기

위 무리를 3년 굴려보자(반올림).

```
연도:   0살  1살  2살   합계
n(0) =   20   10    5     35
n(1) =   20   10    4     34
n(2) =  18(=10+2·4)  10(=.5·20)  4(=.4·10)  → (18,10,4)  합 32
n(3) =  ...
```

합계가 35 → 34 → 32… 로 **조금씩 줄고 있다.** 계속 굴리면 매년 곱해지는 배율이 점점 **일정한 값 λ**에 가까워지는데, 이 예에선 λ가 1보다 약간 작아 **무리가 서서히 감소**한다. 만약 출산율이나 생존율이 조금만 높아 λ>1이 되면 정반대로 **기하급수적으로 증가**한다.

**직접 해보기**: 엑셀에서 L을 3×3으로 적고, 매 열에 `=MMULT(L, 이전열)`을 넣으면 몇십 년이든 자동으로 굴러간다. 몇 년 지나면 **세 나이대의 비율(20:10:5 같은)이 거의 고정**되는 게 보이는데, 그게 다음에 설명할 "안정 나이 분포"다.

## 6. 오래 굴리면 — 안정 나이 분포와 λ

레슬리 행렬을 아주 여러 번 곱하면 신기한 일이 생긴다 [해외 6][국내 국내고유값].

1. **나이 비율이 고정된다.** 처음엔 나이 프로필이 출렁여도, 오래 지나면 "0살:1살:2살 = 어떤 고정 비율"로 자리 잡는다. 이걸 **안정 나이 분포(stable age distribution)**라 한다.
2. **전체 수는 매년 λ배.** 그 상태가 되면 무리 전체가 **매년 정확히 λ배**로만 변한다. 이 λ를 **개체군 성장률**이라 한다.

즉 처음의 세부는 잊히고, **λ와 안정 분포**라는 두 가지가 무리의 장기 운명을 지배한다. (수학적으로 λ는 행렬 L의 **최대 고유값**, 안정 분포는 그에 대응하는 **고유벡터**다 — 자세한 건 확장편.) 국내 자료들도 "**고유값·고유벡터가 인구 동태 모델에서 시스템의 장기 거동을 알려준다**"고 정리한다 [국내 고유값][국내 행렬].

**λ 읽는 법**:
- **λ > 1**: 매년 늘어난다 → 번성.
- **λ = 1**: 그대로 유지 → 평형.
- **λ < 1**: 매년 줄어든다 → 결국 멸종.

그래서 보전생물학자는 멸종위기종의 L을 만들어 **λ를 계산**한다. λ<1이면 "이대로면 사라진다"는 경고이고, "어느 나이의 생존율을 얼마나 올리면 λ가 1을 넘는가"를 따져 **보호 정책**을 세운다 [해외 6].

실전 예를 하나 상상해보자. 바다거북은 알에서 갓 깬 새끼의 생존율이 매우 낮고, 다 자란 성체가 되기까지 오래 걸린다. 만약 보호 예산이 한정돼 있다면, "알을 더 많이 지킬까, 아니면 성체(어미)를 그물 사고에서 지킬까?"를 정해야 한다. 레슬리(정확히는 확장인 단계행렬) 모델로 각 시나리오의 λ를 계산해보면, 놀랍게도 **성체 한 마리를 지키는 것이 알 수백 개를 지키는 것보다 λ를 더 크게 올리는** 경우가 많다. 어미 한 마리는 앞으로 수년간 계속 알을 낳지만, 알은 대부분 성체가 되기 전에 죽기 때문이다. 이처럼 레슬리 모델은 **"직관과 다른 최적 정책"**을 숫자로 짚어준다 — 이게 이 도구가 보전 현장에서 사랑받는 이유다 [해외 6][국내 개체군성장].

## 7. 흔한 오해 · FAQ

- **"출산율은 왜 맨 윗줄에만 있나?"** — 새끼는 전부 **0살(첫 칸)**로 태어나기 때문이다. 어느 나이 어미가 낳든 결과는 0살이라 **윗줄로 모인다** [해외 8].
- **"생존율이 왜 대각선 바로 아래인가?"** — 살아남으면 **딱 한 살 더** 먹어 **다음 칸으로 한 칸 내려가기** 때문이다. 두 칸 건너뛰진 않는다 [해외 8].
- **"단순 지수성장(rN)이랑 뭐가 다른가?"** — 지수모델은 무리를 **한 덩어리**로 본다. 레슬리는 **나이 구조**를 살려, 나이별 출산·생존 차이를 반영한다. 그래서 더 현실적이다 [국내 개체군성장].
- **"수컷은?"** — 기본 모델은 보통 **암컷만** 센다(새끼를 낳는 쪽). 성비가 일정하다고 가정한다.
- **"λ는 어떻게 구하나?"** — 손으로는 특성방정식, 실제로는 컴퓨터로 고유값을 계산한다. 나이대가 적으면 종이로도 가능(확장편).
- **"현실에서 출산율·생존율은 고정인가?"** — 아니다. 기후·먹이·질병에 따라 매년 변한다. 기본 레슬리는 이를 고정으로 보는 **단순화**이고, 변동을 반영한 확률적 모델은 확장편에서 다룬다.

## 8. 한눈 요약
- 레슬리 행렬 = 무리를 **나이별 벡터**로 보고, **1년치 규칙(출산=윗줄, 생존=대각선)**을 담은 행렬 L.
- 예측 = **L을 곱할 때마다 1년**. n(t)=Lᵗ·n(0).
- 오래 굴리면 **안정 나이 분포**(비율 고정)로 수렴하고 전체는 매년 **λ배**.
- **λ(최대 고유값)**: >1 번성 / =1 유지 / <1 멸종.
- 쓸모: 보전·어업·인구 추계에서 "이 종이 늘까 줄까"를 판정.

## 더 깊이 — 확장편으로

이해편은 "행렬을 곱하면 미래가 나온다"를 손으로 보여줬다. **왜 최대 고유값이 성장률이고 고유벡터가 안정 분포인지(페론-프로베니우스 정리)**, 특성방정식으로 λ를 실제로 푸는 법, 재생산가치(좌고유벡터), 그리고 **단계구조(레프코비치 행렬)·확률적 변동·민감도 분석**으로 가는 확장은 **[[루이스-레슬리 행렬 확장편]]**에서 다룬다.
