# 행렬의 대각화 — 이해편

> 목표: "**행렬의 대각화가 무엇이고 왜 유용한지**"를, **피보나치 수열의 일반항을 행렬로 구하는** 대표 예제로 납득시키기. 대각화 가능 조건·증명·스펙트럼 정리·조던형은 확장편 [[행렬의 대각화 확장편]] 으로.

## TL;DR
- **대각화**란 정사각행렬 A를 **A = P D P⁻¹** 꼴로 쪼개는 것이다. 여기서 **D는 대각행렬**(대각선 밖은 전부 0), **P는 A의 고유벡터를 열로 모은 행렬**이다 [1(국내)][6].
- 왜 쪼개나? **거듭제곱이 거저 계산되기 때문**이다. **Aⁿ = P Dⁿ P⁻¹** 이고, 대각행렬의 n제곱 Dⁿ은 **대각원소만 각각 n제곱**하면 끝이다 [3(국내)][7].
- 이걸 쓰면 **피보나치 일반항**이 나온다. 점화식 F(n+1)=F(n)+F(n−1)을 행렬 **Q=[[1,1],[1,0]]** 의 곱으로 쓰고, Q를 대각화해 Qⁿ을 구하면 **비네 공식 F(n)=(φⁿ−ψⁿ)/√5**(φ=황금비)가 딱 떨어진다 [국내 yjh0853][8].
- **의미**: 고유벡터를 새 좌표축으로 삼으면, A는 그 축마다 **λ배(고유값)로 늘이는 단순한 변환**으로 보인다. 복잡한 행렬을 "**각 방향의 배율**"로 해체하는 것 [2(국내)][6].
- **방법 4단계**: ①고유값(det(A−λI)=0) ②고유벡터(각 λ) ③P·D 조립 ④A=PDP⁻¹. 고유벡터가 **n개 독립**이면 대각화된다 [국내 camposcoffee][6].

## 핵심 포인트
1. **대각화 = 행렬의 인수분해.** 복잡한 A를 P·D·P⁻¹ 세 조각으로 [국내 yjh0853].
2. **핵심 이득 = 거듭제곱.** Aⁿ = P Dⁿ P⁻¹, Dⁿ은 공짜 [3(국내)].
3. **P의 열 = 고유벡터, D의 대각 = 고유값.** 이 대응만 외우면 된다 [1(국내)].
4. **피보나치가 대표 응용.** 행렬 Qⁿ을 대각화로 구해 일반항 유도 [국내 yjh0853].
5. **항상 되는 건 아니다.** 독립 고유벡터가 n개 있어야 대각화된다(확장편) [국내 camposcoffee].

## 1. 왜 필요한가 — "A를 100번 곱하기 싫다"

행렬 A가 있을 때 **Aⁿ**(A를 n번 곱한 것)을 자주 구해야 한다. 인구가 매년 A만큼 변하면 n년 뒤는 Aⁿ, 점화식을 n번 적용해도 Aⁿ이 나온다(앞서 다룬 [[레슬리 행렬]]도 이 구조였다). 그런데 **행렬 곱을 100번 하는 건 끔찍**하다. 숫자가 뒤엉켜 손으로는 불가능에 가깝다 [3(국내)].

그런데 **대각행렬**이라면 이야기가 다르다. 예를 들어
```
D = ┌ 2  0 ┐   이면   Dⁿ = ┌ 2ⁿ  0 ┐
    └ 0  3 ┘              └ 0  3ⁿ ┘
```
**대각원소를 각각 n제곱**하면 끝이다. 곱이 서로 안 섞이니까. 그래서 자연스러운 질문이 생긴다. **"보통 행렬 A도 어떻게든 대각행렬처럼 다룰 수 없을까?"** 그 답이 **대각화**다 [7][국내 skkong89].

배울 점: **어려운 계산(Aⁿ)을 쉬운 계산(Dⁿ)으로 바꾸는 '좌표 갈아타기'** 가 대각화의 정신이다. 수학에서 "**어려운 문제를 좌표(관점)를 바꿔 쉬운 문제로 만든다**"는 발상은 도처에 나오는데, 대각화는 그중 가장 깔끔하고 쓸모 많은 사례다.

## 2. 고유벡터 — A가 방향을 안 바꾸는 특별한 화살표

대각화의 열쇠는 **고유벡터**다. 대부분의 화살표 v는 행렬 A를 곱하면 **방향이 휙 바뀐다**. 그런데 어떤 특별한 화살표는 **방향은 그대로고 길이만 λ배**로 변한다:
```
A v = λ v      (v ≠ 0)
```
이때 v를 **고유벡터**, λ를 그 **고유값**이라 한다 [2(국내)][6]. "A를 곱해도 자기 방향을 고집하는" 화살표인 셈이다.

**핵심 아이디어**: 이런 고유벡터들을 **새 좌표축**으로 삼으면, A는 그 축을 따라 **그냥 λ배 늘이는 것**이 된다. x축 방향은 λ₁배, y축 방향은 λ₂배… 이렇게 **각 축이 독립적으로 늘어나는** 게 바로 대각행렬 D다. 그래서 **"고유벡터 좌표에서 보면 A가 대각행렬로 보인다"**는 게 대각화의 그림이다 [6][국내 aria1th].

## 3. 대각화의 정의 — A = P D P⁻¹

이제 정의를 쓴다. 정사각행렬 A가 **대각화 가능**하다는 건, 어떤 **가역행렬 P**와 **대각행렬 D**가 있어
```
A = P D P⁻¹      (똑같이:  P⁻¹ A P = D)
```
가 성립하는 것이다 [1(국내)][6]. 조각의 뜻은 이렇다:
- **P**: A의 **고유벡터를 열(column)로 나란히** 세운 행렬.
- **D**: 그 고유벡터에 대응하는 **고유값을 대각선에** 놓은 대각행렬(순서는 P의 열 순서와 맞춘다).
- **P⁻¹**: P의 역행렬. "새 좌표 ↔ 원래 좌표"를 오가는 통역.

**읽는 법**: A를 곱하는 건 = ①P⁻¹로 **고유벡터 좌표로 갈아타고** → ②D로 **각 축을 λ배 늘이고** → ③P로 **원래 좌표로 돌아오는** 세 단계다. 국내 자료는 이를 "행렬의 **인수분해**"라고 부른다 [국내 yjh0853].

```
    A                =    P      ·   D    ·   P⁻¹
(복잡한 변환)          (고유벡터로   (축마다   (원좌표로
                        갈아타기)    λ배)      되돌리기)
```

## 4. 왜 거듭제곱이 쉬워지나 — Aⁿ = P Dⁿ P⁻¹

대각화의 진짜 보상이 여기다. A = PDP⁻¹ 이면:
```
A² = (PDP⁻¹)(PDP⁻¹) = P D (P⁻¹P) D P⁻¹ = P D² P⁻¹
A³ = P D³ P⁻¹
⋯
Aⁿ = P Dⁿ P⁻¹
```
중간의 **P⁻¹P가 단위행렬 I로 사라지면서**, A를 n번 곱하는 일이 **Dⁿ 하나**로 압축된다. 그리고 Dⁿ은 §1에서 봤듯 **대각원소만 n제곱**하면 끝이다 [3(국내)][7].

즉 **"A를 n번 곱하기" → "P·Dⁿ·P⁻¹ 곱 세 번"** 으로 바뀐다. n이 아무리 커도 계산량이 거의 그대로다. 이게 대각화가 강력한 이유이고, 다음 절 **피보나치**가 그 화려한 예다.

이 트릭이 왜 통하는지 한 번 더 음미하자. 보통 좌표에서 A를 곱하면 **각 성분이 서로 뒤엉킨다**(가로세로가 섞인다). 그런데 고유벡터 좌표로 갈아타면 그 **엉킴이 풀려**, 각 축이 서로를 건드리지 않고 **독립적으로 λ배씩** 자란다. 서로 간섭하지 않으니 n번을 곱해도 그냥 각 축이 λⁿ배가 될 뿐이다. **대각화는 "얽힌 실을 풀어 나란히 세우는" 작업**이라고 보면 된다 — 한번 풀어두면(P⁻¹), 아무리 반복해도(Dⁿ) 다시 얽히지 않고, 마지막에 원래 모양으로 되감기(P)만 하면 된다. 이 "**분리(decoupling)**"가 대각화가 주는 근본 선물이고, 점화식·미분방정식·확률과정에서 똑같이 쓰인다.

## 5. Worked Example — 피보나치 일반항을 행렬로 구하기

**목표**: F(0)=0, F(1)=1, F(n+1)=F(n)+F(n−1)인 피보나치 수열의 **일반항 F(n)**을 구한다.

### (1) 점화식을 행렬로
연속한 두 항을 벡터로 묶고, 한 걸음 나아가는 규칙을 행렬로 쓴다:
```
┌ F(n+1) ┐   ┌ 1  1 ┐ ┌ F(n)   ┐
│ F(n)   │ = │ 1  0 │ │ F(n−1) │
└        ┘   └      ┘ └        ┘
```
(윗줄: F(n+1)=1·F(n)+1·F(n−1) ✓ / 아랫줄: F(n)=1·F(n)+0 ✓). 이 행렬을 **Q = [[1,1],[1,0]]** 라 하자. 그러면 한 걸음이 Q 한 번이므로:
```
┌ F(n+1) ┐        ┌ F(1) ┐        ┌ 1 ┐
│ F(n)   │ = Qⁿ · │ F(0) │ = Qⁿ · │ 0 │
└        ┘        └      ┘        └   ┘
```
**결국 Qⁿ만 구하면 F(n)이 나온다.** 여기서 대각화가 등장한다.

### (2) Q의 고유값 — 황금비 등장
det(Q − λI) = 0 을 푼다:
```
det ┌ 1−λ   1  ┐ = (1−λ)(−λ) − 1·1 = λ² − λ − 1 = 0
    └  1   −λ  ┘
⇒ λ = (1 ± √5) / 2
```
두 고유값을 **φ = (1+√5)/2 ≈ 1.618(황금비)**, **ψ = (1−√5)/2 ≈ −0.618** 이라 하자. (φ+ψ=1, φ−ψ=√5, φψ=−1 — 뒤에서 쓴다.)

### (3) 고유벡터·대각화
φ²=φ+1 이므로 Q·(φ,1)ᵀ = (φ+1, φ)ᵀ = (φ², φ)ᵀ = φ·(φ,1)ᵀ. 즉 **(φ,1)이 φ의 고유벡터**, 같은 식으로 **(ψ,1)이 ψの 고유벡터**다. 따라서
```
P = ┌ φ  ψ ┐,   D = ┌ φ  0 ┐,   Q = P D P⁻¹
    └ 1  1 ┘        └ 0  ψ ┘
```

### (4) Qⁿ과 일반항
```
Qⁿ = P Dⁿ P⁻¹ = P ┌ φⁿ  0  ┐ P⁻¹
                  └ 0   ψⁿ ┘
```
P⁻¹ = 1/(φ−ψ) · [[1,−ψ],[−1,φ]] = 1/√5 · [[1,−ψ],[−1,φ]] 를 넣고 (2,1)성분(=F(n))만 뽑아 정리하면:
```
        φⁿ − ψⁿ       φⁿ − ψⁿ
F(n) = ────────── = ──────────      ← 비네(Binet) 공식
        φ − ψ           √5
```
**검산**: F(5) = (φ⁵−ψ⁵)/√5. φ⁵≈11.09, ψ⁵≈−0.09 → (11.09+0.09)/2.236 ≈ **5** ✓ (실제 수열 0,1,1,2,3,**5**,8,…에서 F(5)=5).

놀라운 점: **무리수 φ·ψ가 잔뜩 들어간 식인데 결과는 항상 정수**다. ψ≈−0.618이라 ψⁿ이 0에 빠르게 가까워져, 사실 **F(n) ≈ φⁿ/√5의 반올림**으로도 구해진다. "피보나치가 황금비로 자란다"는 유명한 사실이 여기서 수식으로 증명된다 [국내 yjh0853][8].

## 6. 흔한 오해 · FAQ

- **"모든 행렬이 대각화되나?"** — 아니다. **독립인 고유벡터가 n개**(n×n행렬 기준) 있어야 P를 만들 수 있다. 고유값이 전부 서로 다르면 항상 되고, 중복되면 안 될 수도 있다(확장편) [국내 camposcoffee][6].
- **"왜 P⁻¹까지 곱하나?"** — P는 "고유벡터 좌표로 갈아타는" 통역이고, 계산이 끝나면 **P⁻¹로 원래 좌표로 돌아와야** 하기 때문이다(§3) [1(국내)].
- **"D의 순서는 상관없나?"** — P의 열 순서와 D의 대각 순서만 **서로 맞으면** 어떤 순서든 된다. φ를 먼저 놓든 ψ를 먼저 놓든 최종 결과는 같다(단, P와 D를 짝 맞춰 바꿔야 한다).
- **"피보나치는 정수인데 왜 무리수 식?"** — φ, ψ가 켤레(짝)라서 무리수 부분이 상쇄된다. 결과는 항상 정수 [8].
- **"레슬리 행렬과 무슨 관계?"** — 똑같다. 레슬리 행렬도 Lⁿ의 장기 거동을 **최대 고유값**이 지배했다([[레슬리 행렬]]). 피보나치의 φ가 바로 그 "성장률"에 해당한다.
- **"고유값이 복소수면?"** — 이차식이 실근을 안 가지면 고유값이 복소수가 된다(예: 회전행렬). 그래도 복소수 범위에서 대각화가 가능하고, 결과는 진동(sin·cos)으로 해석된다 — 실수 점화식에서 진동하는 수열이 나오는 이유다(확장편).
- **"φ가 1보다 크다는 게 왜 중요?"** — φ>1이라 φⁿ이 폭발적으로 커진다. 그래서 피보나치가 **기하급수적으로 증가**한다. 만약 두 고유값이 모두 1보다 작았다면 수열은 0으로 사라졌을 것이다. **고유값의 크기가 수열의 운명을 결정**한다 — 이 통찰이 레슬리 행렬의 λ 판정과 정확히 같다.
- **"고유값·고유벡터를 매번 손으로 구해야 하나?"** — 개념 이해와 2×2·3×3 손계산에는 위 4단계가 정본이지만, 큰 행렬이나 실무에서는 컴퓨터가 대신 구한다(예: 파이썬 `numpy.linalg.eig(A)` 한 줄이면 고유값·고유벡터를 돌려준다). 손계산의 목적은 "왜 그렇게 되는지"를 몸으로 아는 것이고, 실전 계산은 도구에 맡기면 된다. 다만 결과 해석 — **어느 고유값이 가장 크고 그 방향이 무엇을 뜻하는지** — 은 사람이 해야 하며, 그게 이 글이 겨냥한 감각이다.
- **"대각화하면 정확히 뭐가 이득인가, 한 문장으로?"** — "**행렬을 곱하는 어려운 일**을, 고유벡터 좌표로 갈아타서 **숫자 몇 개를 각각 거듭제곱하는 쉬운 일**로 바꾼다." 거듭제곱·점화식·미래 예측이 전부 이 한 줄의 응용이다.

## 7. 한눈 요약
- 대각화 = **A = P D P⁻¹**(P=고유벡터 열, D=고유값 대각). 행렬의 인수분해.
- 이득 = **Aⁿ = P Dⁿ P⁻¹**, Dⁿ은 대각원소 n제곱으로 공짜.
- 방법 = ①고유값 det(A−λI)=0 ②고유벡터 ③P·D 조립 ④검산.
- 피보나치 = Q=[[1,1],[1,0]] 대각화 → **비네 F(n)=(φⁿ−ψⁿ)/√5**.
- 조건 = **독립 고유벡터 n개**면 대각화 가능(아니면 확장편의 조던형). 고유값의 크기가 수열의 증가·소멸을 지배한다.

## 더 깊이 — 확장편으로

이해편은 "대각화가 뭐고 왜 좋은지"를 피보나치로 보여줬다. **정확히 언제 대각화가 가능한지(대수적·기하적 중복도), 대칭행렬은 항상 직교대각화된다는 스펙트럼 정리, 대각화가 안 될 때의 조던 표준형, 그리고 선형 미분방정식·마르코프 연쇄·행렬지수·PCA로 뻗는 응용**은 **[[행렬의 대각화 확장편]]**에서 다룬다.
