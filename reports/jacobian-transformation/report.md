# 야코비안 변환(Jacobian transformation) — 변수변환의 핵심 도구 심층 분석

## TL;DR
- **야코비 행렬(Jacobian matrix)**은 여러 변수를 입력받아 여러 값을 내놓는 벡터함수의 **1차 편도함수를 모아놓은 행렬**이다. 그 정사각 행렬의 **행렬식이 '야코비안(Jacobian determinant)'**이며, 학부 미적분에서는 흔히 이 행렬식 자체를 야코비안이라 부른다 [1][7][4].
- 핵심 직관 하나면 끝난다: **야코비안 행렬식 = 변환이 국소적으로 넓이/부피를 늘리거나 줄이는 배율**. 행렬식이 2면 그 점 근처 미소 면적이 2배로 늘어나고, 0.5면 절반으로 줄어든다 [7][5].
- 그래서 **다중적분의 변수변환(치환적분)**에서 `dx dy = |J| du dv` 처럼 야코비안 절댓값을 곱해 좌표 왜곡을 보정한다. 대표 예가 극좌표: `dx dy = r dr dθ` (여기서 J = r) [8][9][10].
- 응용은 적분에 그치지 않는다 — **확률밀도함수 변환**(p_Y(y)=p_X(x)|det J|), **로보틱스 미분기구학**(관절속도 ↔ 말단속도 사상), **역함수 정리**(국소 가역성 판정)까지 같은 원리가 관통한다 [11][12][2].
- **`det J = 0`인 점은 위험신호**다: 변환이 그 점에서 국소적으로 가역적이지 않고(차원 붕괴), 변수들이 독립이 아니며, 로봇이라면 **특이점(singularity)**에 해당한다 [3][2].

## 핵심 포인트
1. **행렬 vs 행렬식 구분**: 야코비'행렬'(직사각형 가능)은 국소 선형근사 그 자체이고, 야코비'안(행렬식)'은 정사각일 때만 정의되며 부피 배율을 뜻한다 [1][7].
2. **국소 선형화**: 비선형 변환도 한 점 근처를 확대하면 선형변환처럼 보이는데, 그 선형변환을 나타내는 행렬이 바로 야코비 행렬이다 [5][7].
3. **변수변환의 보정자**: 1변수 치환 `dx = (dx/dt)dt`의 다변수 일반화가 `dV → |det J| dV`. 좌표계를 바꾸면 격자 칸 크기가 변하므로 반드시 보정해야 한다 [4][8].
4. **정변환 vs 역변환 주의**: 적분은 보통 `(x,y)→(u,v)` 방향 J를, 확률밀도 변환은 종종 **역변환의 J**를 쓴다. 어느 방향인지 헷갈리면 부호·역수 실수가 난다 [11][2].
5. **유래**: 독일 수학자 **카를 구스타프 야코프 야코비(C. G. J. Jacobi, 1804–1851)**의 이름에서 왔다 [3].

## 본문

### 1. 큰 그림 — 야코비안은 어떤 질문에 답하나
다변수 함수 `F: ℝⁿ → ℝᵐ`, 즉 여러 입력을 받아 여러 출력을 내는 변환을 생각하자. 우리가 자주 던지는 질문은 이렇다:
- 이 변환은 한 점 근처에서 **얼마나, 어느 방향으로 늘이고 비트는가?**
- 좌표계를 `(x,y)`에서 `(u,v)`로 바꾸면 **면적/부피는 얼마나 왜곡되는가?**
- 이 변환은 그 점에서 **되돌릴 수 있는가(가역인가)?**

이 세 질문에 **하나의 객체**로 답하는 것이 야코비안이다. 비결은 "비선형이라도 아주 좁게 보면 선형"이라는 미적분의 근본 아이디어다 [5][7].

### 2. 정의 — 야코비 행렬과 야코비 행렬식
변환 `F(x₁,…,xₙ) = (f₁,…,fₘ)`에 대해, **야코비 행렬**은 모든 1차 편도함수를 모은 `m×n` 행렬이다 [1][7]:

```
        ⎡ ∂f₁/∂x₁  ∂f₁/∂x₂  …  ∂f₁/∂xₙ ⎤
  J  =  ⎢ ∂f₂/∂x₁  ∂f₂/∂x₂  …  ∂f₂/∂xₙ ⎥
        ⎢   ⋮         ⋮     ⋱     ⋮     ⎥
        ⎣ ∂fₘ/∂x₁  ∂fₘ/∂x₂  …  ∂fₘ/∂xₙ ⎦
```

- 입력·출력 차원이 같을 때(`m = n`, 정사각)만 **행렬식**을 취할 수 있고, 이 값 `det J`를 **야코비안(야코비 행렬식)**이라 부른다 [7][2].
- 2변수의 표준형(적분에서 가장 자주 봄): `x=x(u,v), y=y(u,v)`일 때

```
  ∂(x,y)         | ∂x/∂u   ∂x/∂v |
  ─────  = det   |               | = (∂x/∂u)(∂y/∂v) − (∂x/∂v)(∂y/∂u)
  ∂(u,v)         | ∂y/∂u   ∂y/∂v |
```
[8][9]

> **용어 정리(혼동 주의)**: "야코비안"이 (a) 행렬을 가리킬 때도, (b) 행렬식을 가리킬 때도 있다. 한국 학부 미적분에서는 **'야코비 행렬식'을 그냥 야코비안**이라 부르는 관행이 강하다 [4]. 본문은 행렬은 "야코비 행렬", 행렬식은 "야코비안/det J"로 구분한다.

### 3. 기하학적 의미 — 국소 선형근사와 넓이/부피 배율
야코비안을 이해하는 가장 빠른 길은 **기하**다. 비선형 변환이라도 한 점 `p` 주변을 충분히 확대하면 거의 선형변환으로 보이는데, **그 선형변환의 행렬이 바로 그 점에서의 야코비 행렬** `J(p)`다 [5][7].

- 선형변환에서 "넓이/부피가 변하는 배율 = 행렬식의 절댓값"이다. 따라서 **`|det J(p)|`는 변환이 점 `p` 근처에서 미소 넓이/부피를 늘리거나 줄이는 국소 배율**이 된다 [7][5].
- `det J = 2` → 그 점 근방 미소 면적이 약 2배로 팽창, `det J = 0.5` → 절반으로 수축 [7].
- **부호**까지 보면 방향(상대적 향, orientation)의 보존/반전을 알려준다. 그래서 적분에서는 넓이의 크기만 필요하므로 **절댓값** `|det J|`를 쓴다 [8][9].

이 한 문장 — *"야코비안 = 국소 부피 배율"* — 이 아래 모든 응용을 관통한다.

### 4. 응용 ① 다중적분의 변수변환 (극좌표 완결 예제)
1변수 치환적분을 떠올리자: `x=g(t)`로 바꾸면 `dx = g'(t)dt`처럼 **미분소에 도함수가 곱해진다**. 다변수에서는 이 도함수 자리에 **야코비안 절댓값**이 들어온다 [8][4]:

```
  ∬_R f(x,y) dx dy = ∬_S f(x(u,v), y(u,v)) · |∂(x,y)/∂(u,v)| du dv
```
여기서 `R`은 원래 영역, `S`는 새 좌표에서의 대응 영역이다 [9][10].

**왜 곱해야 하나(직관)**: 좌표계를 바꾸면 `du`, `dv`가 만드는 작은 격자 칸이 원래 `xy`평면에서는 더 이상 같은 크기의 직사각형이 아니다. 그 칸의 실제 넓이가 `|det J| du dv`이기 때문에, 보정하지 않으면 넓이를 잘못 세게 된다 [4][8].

**극좌표 예제(가장 중요)**: `x = r cosθ, y = r sinθ` 이면 [8][9][10]

```
  ∂(x,y)   | cosθ   −r sinθ |
  ────── = |                | = r·cos²θ + r·sin²θ = r
  ∂(r,θ)   | sinθ    r cosθ |
```

`r ≥ 0`이므로 `|J| = r`, 따라서 그 유명한 **`dx dy = r dr dθ`**가 나온다. 예컨대 `∬ √(x²+y²) dA`는 극좌표에서 `∬ r · r dr dθ = ∬ r² dr dθ`가 된다(피적분함수의 `r`과 야코비안의 `r`이 곱해짐) [8]. 직관적으로, 극좌표의 "휜 직사각형" 한 칸의 넓이가 `r Δr Δθ`라는 사실과 정확히 일치한다 [9].

### 5. 응용 ② 확률밀도함수(PDF) 변환
확률변수 `X`를 일대일 함수로 변환해 `Y = F(X)`를 얻을 때, `Y`의 밀도는 어떻게 되나? **'국소 부피 배율'로 확률질량을 보정**한다 [11][13]:

```
  p_Y(y) = p_X(x) · |det J|        (단, x = F⁻¹(y))
```

직관: 변환이 공간을 늘이면(`|J|`가 크면) 같은 확률질량이 더 넓게 퍼져 **밀도는 낮아져야** 한다. 그래서 확률이 보존되도록(적분값이 1로 유지되도록) 야코비안으로 곱해 보정한다 — 이를 **'야코비안 보정(Jacobian adjustment)'**이라 부른다 [11].

> **방향 주의**: 보통 `p_Y(y) = p_X(F⁻¹(y)) · |det J_{F⁻¹}(y)|`처럼 **역변환의 야코비안**을 쓴다. 정변환의 야코비안을 쓸 거면 그 **역수**가 들어간다(`|det J_F|` 와 `|det J_{F⁻¹}|`는 서로 역수). 이 방향을 헷갈리면 결과가 뒤집힌다 [11][2]. 현대 머신러닝의 **정규화 흐름(normalizing flows)**·변분추론도 바로 이 보정에 기반한다 [11].

### 6. 응용 ③ 로보틱스 — 미분기구학(속도 사상)
로봇팔에서 **야코비 행렬은 관절속도(joint velocity)와 말단장치속도(end-effector velocity)를 잇는 선형사상**이다 [12]:

```
  ẋ = J(q) · q̇      (q: 관절각, ẋ: 말단의 선·각속도)
```

- 야코비 행렬의 **각 열(column)은 하나의 관절**에 대응하며, 그 관절을 움직일 때 말단이 어떻게 움직이는지를 나타낸다 [12].
- **쌍대성(duality)**으로 같은 `J`가 **관절토크 ↔ 말단힘**도 연결한다(`τ = Jᵀ F`) [12].
- 여기서 `det J = 0`(혹은 비정사각이면 랭크 저하)인 자세가 **특이점(singularity)**이다: 어떤 방향으로는 말단을 아무리 관절을 빨리 돌려도 움직일 수 없거나, 역으로 역기구학 해가 발산한다 [12][3].

### 7. 역함수 정리와 `det J = 0`의 의미
야코비안은 **"이 변환을 국소적으로 되돌릴 수 있는가?"**를 판정한다.
- **역함수 정리(Inverse Function Theorem)**: 어떤 점에서 `det J ≠ 0`이면, 그 점 근방에서 변환은 **국소적으로 가역**이고 매끄러운 역함수를 가진다 [2][7].
- 반대로 **`det J = 0`**이면: 그 점에서 변환이 차원을 '눌러' 넓이를 0으로 만들고(국소 비가역), **변수들이 서로 독립이 아님**을 뜻한다 [3]. 적분 치환에서는 전단사(일대일·온투) 전제가 깨지는 위험 지점이고, 로봇에서는 특이점이다 [2][12].

> **연결 고리**: 야코비안의 부호·영(零) 여부 하나가 *적분 가능성·확률 보존·로봇 제어 안정성*을 동시에 좌우한다. 이것이 야코비안이 여러 분야에서 같은 얼굴로 반복 등장하는 이유다.

### 8. 비교로 정리하기
| 구분 | 1변수 치환적분 | 다변수 야코비안 변환 |
|---|---|---|
| 보정 인자 | `dx = g'(t) dt` (도함수) | `dV = |det J| dV'` (행렬식 절댓값) [8][4] |
| 대상 | 길이(1D) 스케일 | 넓이·부피(nD) 스케일 [7] |
| 가역 판정 | `g'(t) ≠ 0` | `det J ≠ 0` (역함수 정리) [2] |
| 실패 지점 | `g'=0` | `det J = 0`(특이점/차원붕괴) [3][12] |

| 헷갈리는 쌍 | 무엇 | 주의 |
|---|---|---|
| 행렬 vs 행렬식 | J(행렬) ↔ det J(스칼라) | 행렬식은 정사각(m=n)에서만 [1][7] |
| 정변환 vs 역변환 | J_F ↔ J_{F⁻¹} | 둘은 역수 관계, PDF 변환서 방향 주의 [11][2] |
| 부호 vs 절댓값 | det J ↔ |det J| | 적분/밀도는 절댓값, 향(orientation)엔 부호 [8] |

### 9. 흔한 오해와 실수 (실전 체크리스트)
- ❌ **절댓값 빼먹기**: 적분·밀도 변환에서는 `|det J|`. 부호 그대로 쓰면 음수 넓이/음수 확률이 나온다 [8][9].
- ❌ **방향 혼동**: PDF 변환에서 정변환·역변환 야코비안을 섞어 쓰면 `|J|` 대신 `1/|J|`가 들어가 결과가 뒤집힌다 [11].
- ❌ **영역 변환 누락**: 변수만 바꾸고 적분 영역(`R→S`)과 적분 구간을 새 좌표로 바꾸지 않는 실수 [10].
- ❌ **특이점 간과**: `det J = 0`인 점(예: 극좌표의 원점 `r=0`)에서의 전단사 붕괴를 무시하면 안 된다 [3].
- ✅ **검산 팁**: 단순 선형변환(예: 스케일 2배)으로 `|det J|`가 예상 배율과 맞는지 먼저 확인하면 부호/역수 실수를 빨리 잡는다 [5].
