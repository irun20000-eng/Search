# 구면기하학의 코사인법칙과 곡률 — 이해편

> 목표: "**구면(공)의 표면에서 삼각형은 어떻게 다루고, '휘어짐(곡률)'이란 무엇인지**"를 지구 위 두 도시의 거리 계산이라는 실전 예제로 납득시키기. 제1·제2 기본형식·가우스 사상·테오레마 에그레기움·가우스-보네 정리 같은 미분기하학의 엄밀한 도구는 확장편 [[구면기하학의 코사인법칙과 곡률 확장편]] 으로.

## TL;DR
- **구면 코사인법칙**은 공 표면 위 삼각형의 "변과 각"을 잇는 공식이다: **cos c = cos a·cos b + sin a·sin b·cos C**. 여기서 변 a, b, c는 길이가 아니라 **대원호가 구 중심에서 벌리는 각(중심각, 라디안)**이다 [1(국내)][6].
- 실제 거리는 그 중심각 c에 반지름을 곱한 **d = R·c** 다. 그래서 이 한 식이 **지구 위 두 지점의 최단거리(대원거리)를 구하는 항법·천문학의 핵심 도구**가 된다 [국내 수학산책][9].
- **곡률**은 "얼마나 휘었나"를 재는 수다. 곡선에서는 **κ = 1/R**(반지름 R인 원의 곡률은 1/R, 직선은 0) [3(국내)]. 급히 꺾일수록 크다.
- **곡면**에서는 방향마다 휘는 정도가 달라, 가장 크게·작게 휘는 두 값(**주곡률 κ₁, κ₂**)을 잡는다. 이 둘의 **곱이 가우스 곡률 K = κ₁·κ₂**, 평균이 평균곡률 H다 [국내 happy_yoonssam][7].
- **구는 K = 1/R² > 0**(어디서나 양으로 휨), **평면·원기둥은 K = 0**(안 휘거나 한 방향만 휨), **안장은 K < 0**(반대로 휨). 구면 삼각형의 **내각 합이 180°를 넘는** 이유가 바로 이 양의 곡률이다 [6][국내 가우스곡률].

## 핵심 포인트
1. **구면 삼각형의 변 = 중심각.** 길이 대신 각(라디안)으로 잰다. 이게 평면과 첫 번째 차이 [1(국내)].
2. **코사인법칙 cos c = cos a cos b + sin a sin b cos C.** 세 변 중 둘과 낀 각을 알면 나머지 변이 나온다 [6].
3. **대원거리 = R × 중심각.** 지구 두 점의 최단거리가 이 공식으로 [국내 수학산책].
4. **곡률 = 휘어짐의 척도.** 곡선은 1/R, 곡면은 두 주곡률의 곱(가우스 곡률) [3(국내)][7].
5. **곡률 부호가 기하를 결정.** 양(구)=삼각형 내각합>180°, 0(평면)=180°, 음(안장)=<180° [6].

## 1. 왜 필요한가 — "평면 삼각법이 안 통하는 세계"

지도 위에서 두 도시의 거리를 자로 재면 틀린다. 지구는 **공(구면)**이라, 서울에서 뉴욕까지 가는 최단길은 지도 위 직선이 아니라 **북극 쪽으로 휘는 곡선**이다. 실제 비행기가 그 길로 난다. 이런 "휘어진 면 위의 삼각형"을 다루는 수학이 **구면삼각법(spherical trigonometry)** 이다 [국내 천문학백과][8].

평면삼각법과의 결정적 차이 두 가지:
- **직선이 없다.** 구면에서 두 점 사이의 최단경로는 **대원(great circle)**의 호다. 대원이란 **구의 중심을 지나는 평면이 구면을 자를 때 생기는 원**(예: 적도, 자오선)이다. 그보다 작은 원(위선 등)을 따라가면 더 멀다 [국내 crane206265][9].
- **변을 각으로 잰다.** 구면 삼각형의 변은 대원의 호인데, 그 길이를 **호가 구 중심에서 벌리는 중심각**으로 표현한다. 반지름이 1인 단위구에서는 호의 길이가 곧 그 중심각(라디안)이라 편리하다 [1(국내)].

배울 점: **공간이 휘면 "직선"과 "거리"의 뜻부터 다시 정의**해야 한다. 구면삼각법은 그 재정의 위에서 평면삼각법을 확장한 것이고, 나중에 이 "휘어짐"을 정량화한 것이 **곡률**이다.

## 2. 구면 삼각형 — 세 대원호로 만든 삼각형

**구면 삼각형**은 구면 위 세 점을 **대원호 세 개**로 이은 도형이다. 각 요소를 이렇게 표기한다 [8][국내 crane206265]:
- **변 a, b, c**: 각 꼭짓점의 맞은편 대원호. 길이가 아니라 **중심각**(라디안). 예: 변 a는 꼭짓점 A의 맞은편 호.
- **각 A, B, C**: 각 꼭짓점에서 두 대원호가 이루는 각(그 점에서의 접선 사이 각).

평면 삼각형과 달리 구면 삼각형에는 놀라운 성질이 있다:
```
평면 삼각형:  A + B + C = 180°   (항상)
구면 삼각형:  A + B + C > 180°   (항상, 클수록 넘침이 커짐)
```
넘치는 양 **E = (A + B + C) − 180°** 를 **구면 초과(spherical excess)** 라 부른다. 뒤(§6)에서 보듯 이 초과가 삼각형의 **넓이**와 곧바로 연결되고, 그 뿌리가 구의 **양의 곡률**이다 [6][국내 가우스-보네].

**예시 직관**: 북극(N)에서 적도로 두 자오선을 내리고, 그 두 발을 적도호로 이으면 삼각형이 된다. 두 자오선이 적도와 만나는 각은 각각 90°, 북극에서의 각도 있으니 세 각 합은 **180°를 확실히 넘는다**. 만약 두 자오선을 경도 90° 벌리면 세 각이 모두 90°라 합이 **270°**, 이 삼각형은 구 표면의 정확히 1/8(팔분구)이다.

## 3. 구면 코사인법칙 — 핵심 공식

구면삼각법의 심장은 **제1 코사인법칙**이다 [6][국내 crane206265]:
```
cos c = cos a · cos b + sin a · sin b · cos C
```
- **뜻**: 두 변 a, b와 그 사이 낀 각 C를 알면 **나머지 변 c**를 구한다. 평면 코사인법칙 c² = a² + b² − 2ab·cos C 의 구면 버전이다.
- **평면과 비교**: 변이 짧을 때(a, b가 작을 때) 근사하면 평면 코사인법칙으로 되돌아온다. 즉 **구면법칙이 더 일반적**이고, 지구를 아주 좁게 보면 평면법칙이 맞아떨어진다.

### 어디서 나오나 — 벡터 내적으로 한 줄
구 중심을 원점에 두고, 삼각형의 세 꼭짓점을 **단위벡터** A⃗, B⃗, C⃗(길이 1)로 본다. 두 꼭짓점을 잇는 변의 중심각은 곧 **두 단위벡터 사이 각**이므로, 내적의 정의에서 [국내 reslieu][6]:
```
cos(변 AB) = A⃗ · B⃗
```
즉 **구면 거리 = 두 단위벡터 내적의 arccos**. 여기에 좌표를 넣고 삼중적을 정리하면 위 코사인법칙이 그대로 유도된다. 핵심 감각은 **"구면의 각 = 벡터 내적"** 하나다.

### 사인법칙도 짝으로
코사인법칙과 함께 **구면 사인법칙**이 쓰인다:
```
sin A / sin a = sin B / sin b = sin C / sin c
```
평면 사인법칙(변을 sin으로 나눔)의 구면 버전으로, **변과 각을 번갈아 구할 때** 코사인법칙과 함께 연립한다 [8][국내 reslieu].

## 4. Worked Example — 서울에서 로스앤젤레스까지 대원거리

**목표**: 위도·경도만으로 두 도시의 **최단(대원) 거리**를 구한다. 이게 구면 코사인법칙의 대표 실전이다 [국내 수학산책][9].

### (1) 북극을 제3의 꼭짓점으로
두 도시 P, Q와 **북극 N**을 꼭짓점으로 삼는 구면 삼각형을 만든다. 이러면 각 변의 중심각을 위도·경도로 바로 쓸 수 있다:
- 변 NP = **북극에서 P까지 각** = 90° − (P의 위도) — 이를 **여위도**라 한다.
- 변 NQ = 90° − (Q의 위도).
- 두 변 사이 낀 각 N = **두 도시의 경도 차** Δλ.

### (2) 값 대입
서울 P ≈ 위도 37.5°N, 경도 127°E / 로스앤젤레스 Q ≈ 위도 34°N, 경도 118°W.
```
여위도  a = 90° − 37.5° = 52.5°   (북극→서울)
여위도  b = 90° − 34°   = 56°     (북극→LA)
낀각    C = 경도차 = 127° − (−118°) = 245° → 360°−245° = 115°  (짧은 쪽)
```

### (3) 코사인법칙으로 중심각
```
cos c = cos a·cos b + sin a·sin b·cos C
      = cos52.5°·cos56° + sin52.5°·sin56°·cos115°
      ≈ (0.609)(0.559) + (0.793)(0.829)(−0.423)
      ≈ 0.340 − 0.278 = 0.062
c = arccos(0.062) ≈ 86.4°
```

### (4) 각을 거리로
지구 반지름 R ≈ 6371 km. 중심각을 라디안으로:
```
c = 86.4° × (π/180) ≈ 1.508 rad
d = R·c ≈ 6371 × 1.508 ≈ 9,600 km
```
실제 서울–LA 대원거리(약 9,600 km)와 잘 맞는다. **위도·경도 네 개 숫자 → 코사인법칙 한 번 → 최단거리**. 항공·항해·GPS·천문 관측이 전부 이 골격을 쓴다 [9][국내 수학산책].

> 참고: c가 아주 작을 때(가까운 두 점)는 cos의 역함수가 반올림 오차를 키워, 실무에서는 수학적으로 동치이면서 수치적으로 안정한 **하버사인 공식(law of haversines)** 을 대신 쓴다. 원리는 같은 구면삼각법이다 [6].

## 5. 곡률 — "얼마나 휘었나"를 재는 수

이제 구면삼각법이 왜 평면과 다른지의 **뿌리**로 내려간다. 그 뿌리가 **곡률(curvature)** 이다.

### (a) 곡선의 곡률: κ = 1/R
평면 위 곡선이 한 점에서 얼마나 급히 꺾이는지를 재는 수가 **곡률 κ**다 [국내 sinjungmath][3]:
- **직선**: 안 꺾이니 κ = 0.
- **반지름 R인 원**: 모든 점에서 똑같이 휘고, **κ = 1/R**. 작은 원(작은 R)은 급히 꺾여 곡률이 크고, 큰 원은 완만해 곡률이 작다.
- 일반 곡선은 점마다, 그 점에 **가장 잘 들어맞는 원(접촉원)** 의 반지름 R를 잡아 κ = 1/R. 이 R를 **곡률반지름**이라 한다.

즉 **곡률과 곡률반지름은 서로 역수**다. "휨의 세기"와 "얼마나 큰 원처럼 굽었나"는 같은 말의 앞뒷면이다.

### (b) 곡면의 곡률: 방향마다 다르다
곡면(2차원)은 한 점에서도 **바라보는 방향마다 휘는 정도가 다르다**. 예를 들어 원기둥 옆면은 **둥근 방향으로는 휘지만(κ = 1/R) 축 방향으로는 안 휜다(κ = 0)**. 그래서 한 점에서 방향을 빙 돌려가며 잰 휨(법곡률) 중 [국내 happy_yoonssam][7]:
- **최대로 휘는 값 = 주곡률 κ₁**
- **최소로 휘는 값 = 주곡률 κ₂**

이 두 값이 그 점의 휨을 대표한다. 이 두 방향(주방향)은 항상 **서로 수직**이다.

### (c) 가우스 곡률과 평균곡률
두 주곡률로 두 가지 대표 곡률을 만든다 [7][국내 가우스곡률]:
```
가우스 곡률   K = κ₁ · κ₂     (곱)
평균 곡률     H = (κ₁ + κ₂)/2  (평균)
```
- **가우스 곡률 K**가 이 보고서의 주인공이다. **곱**이라 부호가 중요하다.
- **평균 곡률 H**는 비눗방울·최소곡면 이론에서 핵심이지만, 여기선 곁가지로 둔다.

## 6. 가우스 곡률로 곡면 분류하기

가우스 곡률 K = κ₁·κ₂ 의 **부호**가 곡면의 성격을 가른다 [6][국내 가우스곡률].

| 곡면 | 주곡률 κ₁, κ₂ | 가우스 곡률 K | 생김새 | 삼각형 내각합 |
|---|---|---|---|---|
| **구(반지름 R)** | 둘 다 1/R, 같은 부호 | **K = 1/R² > 0** | 어디로든 같은 쪽으로 볼록 | **> 180°** |
| **평면** | 0, 0 | **K = 0** | 안 휨 | = 180° |
| **원기둥·원뿔** | 1/R, 0 | **K = 0** | 한 방향만 휨 | = 180° |
| **안장(쌍곡면)** | 반대 부호 | **K < 0** | 한쪽은 위로, 다른쪽은 아래로 | **< 180°** |

핵심 통찰 세 가지:
1. **구는 K = 1/R² > 0.** 반지름이 작을수록(작은 공) 곡률이 크다. 이 양의 곡률이 구면 삼각형 내각합을 180° 넘게 만든다.
2. **원기둥의 K = 0 이 의미심장하다.** 종이를 말아 원기둥을 만들 수 있듯, **원기둥은 찢거나 늘이지 않고 평면으로 펼 수 있다**. 그래서 "본질적 휨"은 0이다. 반면 **구는 아무리 해도 평면으로 못 편다**(그래서 완벽한 세계지도가 불가능하다) [국내 가우스곡률][6].
3. **안장은 K < 0.** 감자칩·말안장 모양. 이 위의 삼각형은 내각합이 180°보다 **작다**. 이것이 **쌍곡기하학**(음의 곡률 세계)이다.

**구면 초과와 넓이의 관계**(맛보기): §2의 구면 초과 E = (A+B+C) − 180° 는 사실 **삼각형 넓이에 곡률을 곱한 값**과 같다. 구(K = 1/R²)에서는
```
넓이 = R² · (A + B + C − π)      (라디안, Girard 정리)
```
즉 **구면 삼각형은 넓을수록 내각합이 더 많이 넘친다**. 팔분구(§2, 세 각 모두 90°)로 검산하면: 넓이 = R²(π/2+π/2+π/2 − π) = R²·(π/2). 구 전체 넓이 4πR²의 정확히 **1/8** ✓. 이 "곡률 × 넓이 = 각의 초과"가 미분기하학의 **가우스-보네 정리**로 일반화되는데, 그 엄밀한 전개는 확장편이 맡는다 [국내 가우스-보네][6].

## 7. 흔한 오해 · FAQ

- **"구면 삼각형 변이 왜 각(라디안)인가?"** — 변은 대원의 호이고, 그 호를 **구 중심에서 벌린 중심각**으로 재는 게 표준이다. 실제 길이는 거기에 R를 곱하면 된다(호 길이 = Rθ). 단위구(R=1)에선 호 길이 = 각이라 편하다 [1(국내)].
- **"평면 코사인법칙이랑 왜 이렇게 다른가?"** — 사실 같은 뿌리다. 구면법칙에서 변이 아주 작으면 cos·sin을 테일러 전개해 평면법칙 c²=a²+b²−2ab cosC 로 수렴한다. **평면은 구의 아주 작은 조각**이다 [6].
- **"곡률이 크면 좋은 건가 나쁜 건가?"** — 가치판단이 아니라 **휨의 세기**일 뿐이다. 곡률반지름 R가 작을수록(급히 굽을수록) 곡률 1/R이 크다 [3(국내)].
- **"원기둥이 왜 K=0인가? 둥근데?"** — 한 방향(축)은 전혀 안 휘어 κ₂=0이라 곱 K=κ₁·0=0. **한 방향이라도 안 휘면 가우스 곡률은 0**이고, 그런 면은 평면으로 펼 수 있다 [국내 가우스곡률].
- **"가우스 곡률과 평균 곡률 중 뭐가 중요한가?"** — 목적에 따라 다르다. **모양이 평면과 근본적으로 다른가(펼 수 있나)** 를 볼 땐 가우스 곡률 K, **비눗막처럼 표면장력이 만드는 모양**을 볼 땐 평균 곡률 H다. 이 글의 주제는 K.
- **"구면 삼각형 내각합은 항상 정확히 얼마?"** — 고정값이 아니다. 작은 삼각형은 180°에 가깝고, 클수록 커진다. **초과분이 넓이에 비례**(넓이 = R²×초과)하기 때문. 무한히 작으면 평면(180°)에 수렴 [6][국내 가우스-보네].
- **"이게 실생활에 쓰이나?"** — 항공·항해 항법(대원항로), GPS 위치계산, 천문학 별 위치, 지도 투영법이 전부 구면삼각법·곡률 위에 선다. "완벽한 평면 지도가 불가능"한 이유도 구의 K≠0이다 [국내 수학산책][9].

## 8. 한눈 요약
- 구면 코사인법칙: **cos c = cos a cos b + sin a sin b cos C** (변=중심각).
- 대원거리: **d = R × c**, c는 코사인법칙으로 구한 중심각. 항법의 핵심.
- 곡률: 곡선은 **κ = 1/R**, 곡면은 두 주곡률 κ₁, κ₂.
- 가우스 곡률 **K = κ₁·κ₂**: 구 1/R²>0, 평면·원기둥 0, 안장 <0.
- 곡률 부호 = 기하: 양이면 삼각형 내각합>180°(구면 초과), 넓이 = R²×초과.
