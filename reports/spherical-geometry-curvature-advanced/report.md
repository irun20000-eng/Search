# 구면기하학의 코사인법칙과 곡률 — 확장편

> 이해편 [[구면기하학의 코사인법칙과 곡률]] 이 "구면 코사인법칙으로 대원거리를 구하고, 곡률이 무엇인지"를 보여줬다면, 확장편은 **곡면을 매개변수로 나타내는 제1·제2 기본형식, 가우스 사상과 형상연산자로 주곡률을 고유값으로 얻는 법, 가우스 곡률의 내재성(테오레마 에그레기움), 측지선·측지곡률, 곡률과 위상을 잇는 가우스-보네 정리, 그리고 구면 코사인법칙이 상수곡률 기하에서 차지하는 위치**를 다룬다.

## TL;DR
- 곡면을 좌표 (u,v)로 매개변수화하면 **제1 기본형식 I = E du² + 2F du dv + G dv²** 가 곡면 위 길이·각·넓이를 재는 **계량(metric)** 이 된다 [국내 ryumochyee][6].
- 곡면이 접평면에서 얼마나 벗어나는지는 **제2 기본형식 II = L du² + 2M du dv + N dv²** 가 잰다. 여기서 **주곡률 κ₁, κ₂ = 형상연산자(shape operator)의 고유값**이다 [국내 jini_go][10].
- **가우스 곡률 K = det(II)/det(I) = (LN − M²)/(EG − F²) = κ₁κ₂.** 구는 K=1/R², 평면·원기둥은 0, 안장은 음 [7][국내 happy_yoonssam].
- **테오레마 에그레기움(놀라운 정리)**: K는 제2 기본형식으로 정의됐지만 **실은 제1 기본형식(계량)만으로 표현된다** → 곡면을 늘이지 않고 구부리면(등장변환) **K는 불변**. K는 **내재적(intrinsic)** 이다 [8][국내 가우스곡률].
- **가우스-보네 정리**: ∬_S K dA + ∮_∂S k_g ds = 2πχ(S). 측지 삼각형에서는 **(A+B+C) − π = ∬ K dA** — 구면 초과가 곧 곡률의 적분이고, 구에선 넓이 = R²·초과(Girard) [9][국내 가우스-보네].

## 핵심 포인트
1. **제1 기본형식 = 계량.** 곡면 내부 길이·각·넓이가 E,F,G로 결정된다 [6].
2. **제2 기본형식 = 접평면 이탈.** 법선 방향 휨을 재고, 형상연산자의 고유값이 주곡률 [10].
3. **K = (LN−M²)/(EG−F²).** 두 기본형식의 판별식 비 [7].
4. **테오레마 에그레기움 = K는 내재적.** 구부려도 안 변함, 구를 평면으로 못 펴는 이유 [8].
5. **가우스-보네 = 곡률적분이 위상(오일러 지표)과 각결손을 지배** [9].

## 1. 직관 압축 (이해편 요약)

구면 코사인법칙 cos c = cos a cos b + sin a sin b cos C 로 대원거리 d=R·c 를 구했고, 곡률은 곡선에서 κ=1/R, 곡면에서 주곡률의 곱 K=κ₁κ₂ 임을 봤다. 구 K=1/R²>0, 평면·원기둥 0, 안장 <0. 확장편은 이 곡률을 **곡면의 매개변수화**로 엄밀히 계산하고, 그것이 왜 **내재적**이며 어떻게 **위상**과 얽히는지로 정밀화한다.

## 2. 곡면의 매개변수화와 제1 기본형식 (계량)

곡면 S를 두 좌표로 나타낸다: **x(u,v) = (x, y, z)**. 각 점에서 접벡터 두 개
```
x_u = ∂x/∂u,   x_v = ∂x/∂v
```
가 **접평면**을 편다. 곡면 위를 걷는 곡선 x(u(t),v(t))의 속도는 x_u u' + x_v v' 이고, 그 길이²(=제1 기본형식)는 [6][국내 ryumochyee]:
```
I = ds² = E du² + 2F du dv + G dv²
E = x_u·x_u,   F = x_u·x_v,   G = x_v·x_v
```
- **의미**: I는 곡면 **내부에서 측정 가능한 모든 것**(호의 길이, 두 곡선이 이루는 각, 영역의 넓이)을 준다. 넓이 원소는 dA = √(EG − F²) du dv.
- **구의 예**(반지름 R, 위경도 (θ,φ)): x = R(sinθ cosφ, sinθ sinφ, cosθ). 계산하면 **E = R², F = 0, G = R² sin²θ**. 그래서 ds² = R²(dθ² + sin²θ dφ²) — 위도에 따라 경도 방향 거리가 sinθ로 줄어드는 것이 계량에 새겨져 있다 [국내 galoise4118][6].

**제1 기본형식이 같으면 = 내재적으로 같은 곡면.** 두 곡면 사이에 I를 보존하는 대응(등장변환, isometry)이 있으면, 그 위 생명체는 둘을 구별하지 못한다. 평면과 원기둥은 국소적으로 등장 — 종이를 말아도 종이 위 거리는 안 변하니까 [국내 ryumochyee 등장변환].

## 3. 가우스 사상과 제2 기본형식

곡면이 3차원 공간에서 **어느 쪽으로 얼마나 휘는지**는 제1 기본형식만으론 부족하다. 각 점의 **단위 법벡터 n**을 본다.

### 가우스 사상
곡면의 각 점에 그 점의 단위법벡터 n을 대응시켜 **단위구 S²로 보내는 사상** N: S → S² 를 **가우스 사상(Gauss map)** 이라 한다. 곡면이 많이 휠수록 n이 빨리 바뀌고, 가우스 사상이 단위구를 빨리 덮는다. **가우스 곡률의 원래 정의**는 이 사상의 국소 넓이 확대율(야코비안)이다 [국내 가우스사상][7].

### 제2 기본형식
접평면에서 곡면이 벗어나는 정도를, 곡면의 이차 도함수를 법선에 사영해 잰다 [10][국내 galoise4118]:
```
II = L du² + 2M du dv + N dv²
L = x_uu·n,   M = x_uv·n,   N = x_vv·n
```
곡면 위 한 접방향으로 자른 곡선의 **법곡률(normal curvature)** 은 두 기본형식의 비로 나온다:
```
κ_n(방향) = II(방향)/I(방향)
```

## 4. 형상연산자 · 주곡률 · 가우스 곡률 계산

**형상연산자(Weingarten map) S = −dN** 는 법벡터의 변화를 접평면 위 선형사상으로 본 것이다. 그 행렬은 두 기본형식으로 **S = I⁻¹ II** (계수행렬 곱). 여기서 [10][국내 jini_go]:
- **주곡률 κ₁, κ₂ = 형상연산자 S의 고유값** (법곡률의 최대·최소).
- **주방향 = 그 고유벡터** (항상 서로 수직) — 이것은 대칭행렬의 대각화이자, [[행렬의 대각화]]에서 본 **고유값=λ, 고유벡터=주축**의 그림 그대로다.

두 대표량은 형상연산자의 **판별식·대각합**으로 [7]:
```
가우스 곡률  K = det(S) = det(II)/det(I) = (LN − M²)/(EG − F²) = κ₁κ₂
평균 곡률    H = ½ tr(S) = (κ₁ + κ₂)/2
```

### Worked 1 — 구의 가우스 곡률 (직접 계산)
반지름 R 구, 위경도 (θ,φ). §2에서 E=R², F=0, G=R²sin²θ. 제2 기본형식 계수를 계산하면 L=R, M=0, N=R sin²θ (법선을 안쪽으로 잡을 때 부호 조정). 따라서
```
K = (LN − M²)/(EG − F²) = (R·R sin²θ − 0)/(R²·R²sin²θ − 0)
  = R² sin²θ / (R⁴ sin²θ) = 1/R²
```
**구는 어디서나 K = 1/R²** — 위도 θ와 무관하게 상수. 이것이 "구는 **상수 양곡률 곡면**"이라는 사실이다 [7][국내 happy_yoonssam].

### Worked 2 — 원기둥·안장 대조
- **원기둥**(반지름 a): 한 주방향 κ₁=1/a, 축 방향 κ₂=0 → **K=0**, H=1/(2a). 가우스 곡률 0이지만 평균 곡률은 0이 아님 → **평면과 내재적으로 같되(펼 수 있다) 공간에는 휘어 있다**.
- **안장**(z = x² − y² 근방 원점): κ₁>0, κ₂<0 → **K<0**. 접평면 양쪽으로 갈라져 벗어나는 쌍곡점 [국내 galoise4118][7].

### Worked 3 — 회전면(원환면·도넛)의 곡률 부호 지도
원환면(torus)은 큰 반지름 R, 관 반지름 r인 도넛이다. 관의 각도 ψ로 매개변수화하면 가우스 곡률은
```
K(ψ) = cos ψ / [ r (R + r cos ψ) ]
```
가 된다. 부호가 위치에 따라 갈린다는 점이 교육적으로 결정적이다:
- **바깥쪽 적도(ψ=0, cos ψ=1 > 0)**: K > 0 — 구처럼 볼록한 타원점.
- **위·아래 꼭대기(ψ=±90°, cos ψ=0)**: K = 0 — 원기둥처럼 한 방향만 휘는 포물점.
- **안쪽 골(ψ=180°, cos ψ=−1 < 0)**: K < 0 — 안장처럼 갈라지는 쌍곡점.

한 곡면 안에 **양·영·음의 곡률이 공존**한다. 그리고 §7 가우스-보네에서 보듯, 바깥쪽 양곡률 총량과 안쪽 음곡률 총량이 정확히 상쇄되어 도넛 전체 ∬K dA = 0(오일러 지표 χ=0)이 된다 — 이 상쇄를 K(ψ) 식을 표면 전체에 적분해 직접 확인할 수 있다 [7][국내 galoise4118].

## 5. 테오레마 에그레기움 — "놀라운 정리"

제2 기본형식(L,M,N)은 **법벡터 n**을 쓴다 — 즉 곡면이 **바깥 공간에 어떻게 놓였는지(외재적, extrinsic)** 에 의존하는 양처럼 보인다. 그런데 가우스가 1827년 증명한 **테오레마 에그레기움(Theorema Egregium, 놀라운 정리)** 은 [8][국내 가우스곡률]:

> **가우스 곡률 K는 제1 기본형식(E, F, G)과 그 도함수만으로 표현된다.**

즉 K는 겉보기 정의(주곡률의 곱=제2형식 관여)와 달리, **곡면 내부에서 거리만 재도 알아낼 수 있는 내재적(intrinsic) 불변량**이다. 결과가 왜 놀라운가:

1. **구부려도 K는 안 변한다.** 늘이지 않고 구부리는 변형(등장변환)은 제1 기본형식을 보존 → K 보존. 종이(K=0)를 말면 원기둥·원뿔(여전히 K=0)은 되지만, **절대 구(K=1/R²)로는 못 만든다**.
2. **완벽한 세계지도는 불가능하다.** 지도는 구(K=1/R²)를 평면(K=0)으로 옮기는 것인데 K가 내재적이라 보존돼야 한다. K가 다르니 **거리·각·넓이를 동시에 보존하는 평면지도는 존재할 수 없다** — 메르카토르(각 보존)·정적도법(넓이 보존)이 서로를 포기하는 근본 이유 [국내 가우스곡률][8].
3. **2차원 생명체도 자기 세계의 곡률을 잴 수 있다.** 바깥 3차원을 못 봐도, 삼각형 내각합이나 원둘레를 재면 K를 안다. 이것이 리만기하학과 일반상대성(중력=시공간 곡률)으로 가는 길목이다.

내재성의 실측 형태(각결손): 반지름 r인 **작은 측지원**의 둘레 C와 넓이 A는 평면값에서 K만큼 어긋난다.
```
C = 2πr (1 − K r²/6 + …),    A = πr² (1 − K r²/12 + …)
```
둘레가 2πr보다 **덜하면 K>0(구)**, **더하면 K<0(안장)**. 자만 있으면 곡률을 안다 [8].

## 6. 측지선 · 측지곡률 — 곡면 위의 "직선"

평면의 직선에 해당하는 곡면 위 "가장 곧은 길"이 **측지선(geodesic)** 이다. 곡면 위 곡선의 곡률을 두 성분으로 쪼갠다 [국내 측지곡률][7]:
- **법곡률 κ_n**: 곡면이 공간에서 휘어서 어쩔 수 없이 생기는 휨(법선 방향).
- **측지곡률 k_g**: 곡선이 곡면 **안에서** 옆으로 휜 정도(접평면 성분). **측지선은 k_g = 0** — 곡면 안에서 보면 전혀 안 휜 "직선".

구에서 측지선은 **대원**이다. 그래서 이해편의 대원거리가 곧 "구면 위 두 점의 측지선 길이"였다. 구면 삼각형은 정확히는 **측지 삼각형**(세 변이 측지선)이다 [국내 crane206265].

**측지선의 변분적 정의**: 측지선은 두 점을 잇는 경로 중 **길이가 극값(대개 최소)** 인 곡선이다. 이 조건을 제1 기본형식(계량)의 라그랑지안 √(E u'² + 2F u'v' + G v'²)에 오일러-라그랑주 방정식으로 걸면 **측지 방정식**이 나온다:
```
u'' + Σ Γ¹_{ij} (좌표속도 곱) = 0,   v'' + Σ Γ²_{ij} (좌표속도 곱) = 0
```
여기 등장하는 **크리스토펠 기호 Γ**는 제1 기본형식 E, F, G의 도함수만으로 계산된다 — 즉 **측지선도 순전히 내재적**이다. 곡면 안의 생명체가 바깥을 못 봐도 "가장 곧은 길"을 정의할 수 있다는 뜻이고, 이것이 일반상대성에서 "자유낙하하는 물체는 시공간의 측지선을 따른다(중력=곡률)"로 이어지는 개념적 다리다 [8][국내 측지곡률].

**물리적 확인 — 진자·초점**: 구면 위에서 측지선(대원)을 따라 벡터를 평행이동시키면, 삼각형을 한 바퀴 돌아왔을 때 벡터가 처음과 **각 결손 = ∬K dA 만큼 회전**해 있다. 푸코 진자의 세차, 그리고 곡률이 만드는 홀로노미(holonomy)가 정확히 이 현상이다 [9].

## 7. 가우스-보네 정리 — 곡률과 위상의 다리

미분기하학의 정점. 곡률의 **국소(local) 적분**이 **전역(global) 위상**과 각결손을 한꺼번에 지배한다 [9][국내 가우스-보네].

### 국소형 (측지 삼각형)
세 변이 측지선인 삼각형 T에서:
```
∬_T K dA = (A + B + C) − π
```
**좌변은 삼각형 내부 곡률의 총량, 우변은 구면 초과**(내각합이 π를 넘는 양). 이해편의 "곡률 × 넓이 = 각 초과"가 정확히 이 식이다.

### Worked 3 — Girard 정리 재유도
구(K=1/R² 상수)에서 좌변 = (1/R²)·(넓이). 그러므로
```
넓이 = R² · (A + B + C − π)      ← Girard 정리
```
**팔분구**(세 각 = π/2): 넓이 = R²(3·π/2 − π) = R²·π/2. 구 전체 4πR²의 1/8 ✓. **삼각형 넓이를 각만으로** 구하는 이 공식은 오직 곡률이 상수라서 가능하다 [9].

### 전역형 (닫힌 곡면)
경계 없는 닫힌 곡면 전체에 대해:
```
∬_S K dA = 2π·χ(S)
```
- **χ(S) = 오일러 지표**(위상 불변량: 구 χ=2, 원환면(도넛) χ=0, 손잡이 g개면 χ=2−2g).
- **구**: ∬ K dA = (1/R²)(4πR²) = 4π = 2π·2 ✓. **곡면을 아무리 찌그러뜨려도(곡률 재분배) K의 총적분은 위상이 못 박아 고정**한다.
- **도넛**: χ=0 이라 총 곡률 = 0 — 바깥쪽 양곡률과 안쪽 음곡률이 정확히 상쇄된다 [국내 가우스-보네][9].

이것이 미분기하학의 경이다: **연속적으로 변하는 곡률의 적분이 정수(위상)로 양자화**된다.

## 9. 구면 코사인법칙의 미분기하학적 위치

이해편의 코사인법칙은 **상수 양곡률 공간(구)의 삼각법**이다. 곡률을 바꾸면 삼각법 자체가 바뀐다 [8][6]:
- **평면(K=0)**: c² = a² + b² − 2ab cos C (유클리드 코사인법칙).
- **구(K=1/R²)**: cos(c/R) = cos(a/R)cos(b/R) + sin(a/R)sin(b/R) cos C. (단위구 R=1이면 이해편 식.)
- **쌍곡면(K=−1)**: cosh c = cosh a cosh b − sinh a sinh b cos C (삼각함수 → 쌍곡함수).

세 법칙은 **곡률 상수 K에 따라 연속적으로 연결**된다. 곡률이 0으로 가는 극한(작은 삼각형)에서 구면·쌍곡 법칙 모두 유클리드 법칙으로 수렴한다. 즉 **코사인법칙은 그 공간의 곡률을 품은 삼각법**이고, 이것이 "구면 코사인법칙을 미분기하학 관점에서 본다"의 핵심이다.

한 걸음 더: 세 경우를 곡률 반지름 ρ = 1/√|K| 로 통일하면, 상수곡률 K인 공간의 코사인법칙은
```
K>0:  cos(c√K) = cos(a√K)cos(b√K) + sin(a√K)sin(b√K) cos C
K<0:  cosh(c√|K|) = cosh(a√|K|)cosh(b√|K|) − sinh(a√|K|)sinh(b√|K|) cos C
```
로 한 가족을 이룬다. **삼각법이 곡률의 함수**라는 이 관점이, 평면 삼각법을 "K=0인 특수한 경우"로 격하시키고 구면·쌍곡을 대등하게 세우는 비유클리드 기하의 통합 시각이다 [11][6].

## 10. 함정 · 체크리스트

- [ ] **제1형식은 내재적, 제2형식은 외재적.** 길이·각·넓이는 I, 공간 속 휨은 II [6][10].
- [ ] **K는 내재적(테오레마 에그레기움), H는 외재적.** 구부리면 H는 변해도 K는 불변 [8].
- [ ] **원기둥 K=0**을 "안 휘었다"로 오해 말 것 — 공간엔 휘어 있으나(H≠0) 내재적으로 평평(펼 수 있음).
- [ ] **주곡률 = 형상연산자 고유값.** 대각화의 언어와 동일, 주방향은 직교 [10].
- [ ] **가우스-보네 좌변은 넓이적분**(K dA), 초과는 라디안. 팔분구로 늘 검산.
- [ ] **측지선 ≠ 항상 최단**: 측지는 국소적으로 곧을 뿐, 대원의 긴 쪽 호도 측지선.
- [ ] **부호 규약(법선 방향)** 에 따라 L,M,N·H의 부호가 뒤집힌다 — K=κ₁κ₂는 부호 무관(둘 다 뒤집힘).
